对比Excel，轻松学习Python报表自动化 · 第5章「用Python设置Excel对齐方式」

SCQA · 情境 → 冲突 → 问题 → 回答 在Excel中调整对齐方式——左对齐、居中、右对齐、自动换行、合并单元格——只需在「开始」选项卡的「对齐方式」组点击几次按钮。但当你有数百行数据需要统一对齐、或每天生成的报表都要标准化排版时，手工调整就枯燥低效。如何用Python批量控制Excel单元格的对齐方式？本章系统讲解Alignment()函数的6大参数、9种水平/3种垂直对齐类型、合并/解除单元格、以及批量对齐的实战案例。

本章聚焦于Excel「开始」选项卡中「对齐方式」组的Python实现。首先介绍Alignment()函数的6个核心参数：horizontal（水平对齐，支持left/center/right/justify/distributed/fill等9种）、vertical（垂直对齐，top/center/bottom）、text_rotation（文本旋转角度，0-360°）、wrap_text（自动换行）、shrink_to_fit（自适应缩小字号）、indent（缩进字符数）。然后讲解单元格合并/解除合并（merge_cells/unmerge_cells），以及合并后单元格通过top_left_cell进行样式设置的方法。最后通过一个批量设置对齐方式加合并标题行的综合案例，展示如何用for循环一次性完成整表标准化排版。

02
水平对齐
水平方向对齐：9种类型从常规到特殊
horizontal参数支持9种对齐类型：left（左对齐）、center（居中）、right（右对齐）、justify（两端对齐，占满行靠两边界）、distributed（分散对齐，未满行自动调字间距占满）、fill（填满对齐，重复字符填充）、general（一般对齐）、centerContinuous、distributed。日常最常用的是left/center/right三种，分散对齐和填满对齐用于特殊排版需求。

03
换行与旋转
自动换行、文本旋转与字号自适应
当文本长度超过列宽时，有三种处理方式：wrap_text=True自动换行让内容在单元格内多行显示；shrink_to_fit=True自动调小字体使内容适配单元格；text_rotation设置文本旋转角度（逆时针）。这三者在实际报表排版中非常实用，特别适合处理长文本标题和备注列。注意text_rotation与shrink_to_fit不能同时使用。

04
合并操作
单元格合并与解除：merge_cells / unmerge_cells
merge_cells('A2:D2')合并指定区域为一个大单元格，对应Excel「合并后居中」按钮。unmerge_cells('A2:D2')解除合并还原。合并后的单元格以左上角单元格（top_left_cell）为代表进行样式设置——字体、填充、对齐方式均作用于top_left_cell，保证合并区域样式一致性。
from openpyxl.styles import Alignment, Font # 合并单元格 ws.merge_cells('A2:D2') # 通过左上角单元格设置合并后样式 top_left_cell = ws['A2'] top_left_cell.alignment = Alignment( horizontal='center', vertical='center') top_left_cell.font = Font(bold=True) # 解除合并 ws.unmerge_cells('A2:D2')

批量设置对齐方式：整表标准化一个循环搞定
真实场景极少只格式化一个单元格。在字体格式化的for循环基础上增加alignment属性，一次性完成水平居中、垂直居中设置。配合标题行合并与加粗，一条for循环完成整表的对齐与排版统一。修改一次代码，永久适用于同类报表。

🔑 核心结论
Excel的对齐方式——水平对齐、垂直对齐、自动换行、文本旋转、缩进、合并单元格——在Python中对应一个核心函数Alignment()和两个操作函数merge_cells()/unmerge_cells()。参数直接映射Excel功能区按钮，学习成本极低。真正的威力在于批量：一个for循环遍历所有数据单元格，统一设置alignment属性，配合标题行合并加粗，让报表排版和数据处理一样全自动化。这就是Python报表自动化的核心价值——取代一切重复、枯燥的工作。
