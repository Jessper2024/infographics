Definition and Essence of Data Analysis

Data analysis uses appropriate tools, supported by statistical theory, to preprocess data and combine it with business context to help departments monitor, locate, analyze, and solve problems—enabling efficient decision-making, improved operational efficiency, and discovery of business opportunities.

Core formula: Tools + Statistics + Preprocessing + Business Context → Effective Decisions + Competitive Advantage. Its essence is extracting hidden insights from massive data, replacing gut-feel decision-making.

  • Beer & Diapers: discovered association from purchase history, enabling bundled sales to boost average order value
  • Google Flu Trends: predicted flu outbreaks through abnormal keyword search volume spikes (concept worth studying)
2
Three Core Functions

Data analysis serves three core functions in daily business operations:

  • Descriptive Analysis — tells you what happened in the past. Monitors KPIs (orders, new users, activity rate, retention rate) with YoY/MoM comparisons to understand overall trends and business composition. Delivered via daily/weekly/monthly reports.
  • Diagnostic Analysis — tells you why a situation exists. Conducts ad-hoc analysis: why did a product's sales suddenly surge? Which region is underperforming? Drills down layer by layer to find root causes.
  • Predictive Analysis — tells you what may happen in the future. Forecasts trends based on historical patterns to inform quarterly/annual planning. Example: stock extra bread before a typhoon hits.
3
Six Metric Dimensions

The emphasis is on "analysis," not on tools. Six dimensions form a complete metric system:

  • Overview Metrics — absolute statistics reflecting overall scale. e.g., daily sales ¥600K, 20K orders, 15K buyers
  • Comparative Metrics — YoY, MoM, and absolute differences. Measure direction and magnitude of change
  • Central Tendency — general level. Arithmetic mean (simple/weighted) and positional mean (median/mode)
  • Dispersion Metrics — data volatility. Range, quartiles, variance, standard deviation
  • Correlation — relationships between variables, measured by coefficient r ∈ [-1,1]. Higher |r| = stronger correlation; sign indicates direction
  • ⚠️ Correlation ≠ Causation — beer and diapers are correlated, flu and keyword search are causal. Always distinguish.
4
Nine-Step Standard Workflow

Data analysis follows a standard nine-step pipeline: Master Tools → Define Purpose → Acquire Data → Explore Data → Clean Data → Analyze Data → Draw Conclusions → Validate Conclusions → Present Results.

  • ① Master Tools — sharpen your axe before chopping wood
  • ② Define Purpose — what conclusions do you want to reach?
  • ③ Acquire Data — identify metrics, time ranges, sources, and extraction methods
  • ④ Explore Data — size, data types, sufficiency (iterative with step ③)
  • ⑤ Clean Data — handle anomalies, duplicates, missing values, test data
  • ⑥ Analyze Data — apply metrics, use drill-down method to pinpoint root causes
  • ⑦ Draw Conclusions — synthesize insights from analysis
  • ⑧ Validate Conclusions — cross-check with real business, test with action
  • ⑨ Present Results — visualize and share with stakeholders
5
Tool Comparison: Excel vs Python

The same operation can be achieved with different tools—Excel through point-and-click, Python through code. Both are data analysis tools.

For example, "importing external data":

  • Excel — Menu: Data → select the format-specific option → done with clicks
  • Python — Write code: choose read_excel() / read_csv() / read_table() based on file format

Book structure: each data analysis operation is demonstrated with both Excel and Python side by side, enabling a smooth transition from familiar Excel thinking to Python programming.

💡 Key Takeaway

Data analysis is not a battle of tools—it's about mastering methodology. Understanding the "3 Functions + 6 Metrics + 9 Steps" framework gives you the universal language of data analysis. Excel and Python are simply two implementation vehicles for this framework—the former lowering barriers through clicks, the latter boosting efficiency and reproducibility through code. Every subsequent chapter follows the "Master Tools → Acquire Data → Clean Data → Analyze Data → Present Results" pipeline, comparing Excel and Python implementations step by step.
