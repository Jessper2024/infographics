Python Introduction & Environment Setup

Python is a glue language with simple syntax and rich libraries, surpassing Java as the #1 programming language in 2018. This book uses the Anaconda distribution as the entry point because it comes with commonly used libraries pre-installed, requiring no manual setup—extremely beginner-friendly.

  • Anaconda Installation: Check system architecture → Download from official site → Choose Python 3 → Double-click to install → Use default path
  • IDE vs IDLE: IDE (Integrated Development Environment) bundles editor + compiler + debugger. Jupyter Notebook is the most popular IDE in the data analysis field
  • Jupyter Notebook Core Ops: Create Python 3 file (New), run code (Ctrl+Enter), rename (File>Rename), save/export (Download as), import files (Upload)
2
Jupyter Notebook Advanced: Markdown & Table of Contents

Jupyter Notebook is not just a code editor—it's an interactive notebook for data analysis. Two advanced features dramatically improve workflow:

  • Markdown Mode: Switch a code cell to Markdown mode to create a text cell supporting Markdown syntax. Write analysis conclusions directly among code cells, achieving "code + documentation" integration
  • Table of Contents: Not built-in; install via Anaconda Prompt with pip install. Once installed, navigate directly to different code sections—similar to book and thesis tables of contents
3
Seven Core Data Types & Operations

Python's seven core data types cover all common data organization forms encountered in data analysis:

  • Numbers — int, float. Support +, -, *, /, exponentiation (**), floor division (//), modulo (%)
  • Strings — concatenation (+), repetition (*), length (len), search (in/find), indexing (regular/slice), split, strip
  • Lists — ordered, mutable collection. Create, copy (*), merge (+ or extend), insert (append/insert), count, index, slice, delete (pop/remove), sort
  • Dictionaries — key-value pairs {key:value}. Keys must be unique. Three core methods: keys(), values(), items()
  • Tuples — ordered, immutable collection, using parentheses. Interconvertible with lists (list()/tuple()). zip() packs multiple iterables into tuples
4
Operators & Control Flow

Four operator categories and three control-flow structures form the core of Python program logic:

  • Arithmetic: +, -, *, /, ** (power), // (floor division), % (modulo)
  • Comparison: ==, !=, >, <, >=, <= — return True/False
  • Logical: and, or, not
  • for Loop — iterate over a sequence (list/string), executing for each item. e.g., learning Excel→SQL→Python→Statistics in order
  • while Loop — execute while condition holds. e.g., study time < 7 weeks → keep studying; ≥ 7 weeks → job-ready
  • Conditionals — if/else/elif. elif chains evaluate sequentially; at most one branch executes
5
Functions & Advanced Features

A function is a reusable block of code. Master both regular and lambda functions, along with list comprehensions and map:

  • Regular Functions: def func_name(params): + body + return. Formal parameters declared in definition, actual arguments passed at call time
  • Lambda Functions: lambda arg1,arg2: expression. More concise than regular functions. e.g., lambda x,y: x+y for adding two numbers
  • List Comprehensions: Replace nested for loops in one line. e.g., [x*y for x in listA for y in listB]. Dramatically faster for large datasets
  • map Function: map(function, args) — applies function to each element. Results require for-loop iteration or list() conversion to display
  • Modules: Upgraded functions, callable across programs. Three core data analysis modules: NumPy, Pandas, matplotlib. Import via import module or from module import func
💡 Key Takeaway

Python fundamentals are the "wok" of data analysis—without a good wok, even the best ingredients won't make a great dish. The Python knowledge system built in this chapter (environment setup → data types → control flow → functions → advanced features) will be used repeatedly in every subsequent chapter. Remember: a data analyst doesn't need to be a full-stack Python engineer. Mastering the fundamentals directly relevant to data processing is enough to begin the analysis journey. Anaconda + Jupyter Notebook = the optimal development environment combo for data analysts.
