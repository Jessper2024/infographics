第1章 · 认识自动化报表

对比Excel，轻松学习Python报表自动化 · 第1章「认识自动化报表」

SCQA · 情境 → 冲突 → 问题 → 回答 Excel作为日常工作核心工具，通过工作簿→Sheet→单元格→元素的层次结构管理数据，每次制作报表都需手动重复操作。但手动制表效率低、易出错，无法满足高频报表需求。如何将重复的人工报表工作转化为自动化流程？答案在于五步自动化流程（拆解→找代码→组合→验证→调用），并选择合适的Python库（推荐Pandas+openpyxl）代替手工操作。

本章是全书的基础框架。首先剖析Excel的底层组成：工作簿包含多个Sheet，每个Sheet由单元格构成，单元格可设置字体、对齐、条件格式等元素。然后介绍了制作自动化报表的五步通用流程——这个流程与工具无关，是方法论层面的抽象。最后梳理了Python操作Excel的八大库（win32com、xlrd、xlwt、xlutils、xlsxwriter、xlwings、openpyxl、Pandas），分析各自的优劣，并确定了本书的技术栈：Pandas负责数据处理，openpyxl负责格式样式设置。

03
库间桥接
Pandas与openpyxl之间的数据转换
实际工作中常需混用Pandas和openpyxl：先用Pandas处理数据，再用openpyxl设置格式。dataframe_to_rows()是连接两个库的桥梁：将DataFrame转化为逐行数据，参数index控制是否保留索引列，header控制是否保留列名。然后用openpyxl的append()将转换后的数据逐行写入工作簿。
# Pandas读取 → openpyxl写入
from openpyxl.utils.dataframe import dataframe_to_rows

df = pd.read_excel("dataset.xlsx")
wb = Workbook()
ws = wb.active

for row in dataframe_to_rows(df, index=False, header=True):
ws.append(row)

wb.save("output.xlsx")

第4章 · 用Python实现单元格选择和字体设置

对比Excel，轻松学习Python报表自动化 · 第4章「用Python实现单元格选择和字体设置」

SCQA · 情境 → 冲突 → 问题 → 回答 在Excel中设置字体格式——选字体、调大小、加粗、变色、填充底色、加边框——只需在功能区点击几个按钮。但当你有数百个单元格需要统一格式、或每天都要对固定区域重复相同格式设置时，手工操作就变得低效易错。如何用Python实现Excel单元格的格式自动化？本章系统讲解了单元格选择（单格/行列/区域）、字体设置（Font()函数的8大参数）、单元格填充（PatternFill()纯色与图案）、以及边框线设置（Border()四边+对角线），最后通过一个批量格式化实战案例将这些知识点串联起来。

本章聚焦于Excel「开始」选项卡中「字体」组的Python实现。首先介绍了单元格的多种选择方式：直接指明坐标（ws['A1']）或行列编号（ws.cell()）选择单格；通过行号、列号、范围选择多个单元格。然后深入Font()函数的8个参数：name（字体类型）、size（大小）、bold（加粗）、italic（斜体）、vertAlign（垂直对齐）、underline（下划线类型）、strike（删除线）、color（ARGB颜色值）。接着讲解了PatternFill()实现单元格填充——fill_type='solid'纯色填充是最常用场景，以及Border()设置四边边框线的线型与颜色。最后通过一个批量格式化案例展示了如何遍历多行多列，统一设置所有单元格的格式。

第8章 · 用Python对Excel进行编辑

对比Excel，轻松学习Python报表自动化 · 第8章「用Python对Excel进行编辑」

SCQA · 情境 → 冲突 → 问题 → 回答 Excel的「开始」选项卡「编辑」组提供排序、筛选、查找替换三大核心功能。手工操作时，选中列→点击排序→选升序/降序，或点击筛选→勾选条件——每个Sheet重复一遍。当数据每天更新、每次都要重新排序筛选时，这些重复操作就是生产力杀手。如何用Python自动完成Excel的排序与筛选？本章讲解用Pandas的sort_values()实现单列与多列排序、用Pandas条件表达式实现数字筛选与文本筛选、以及查找替换的概要介绍。

本章聚焦于Excel「开始」选项卡「编辑」组的Python实现。核心要点：openpyxl的排序和筛选只会添加排序/筛选符号，不会真的改变数据顺序或过滤数据。真正的排序操作需使用Pandas库的sort_values(df, by=col, ascending=False)，支持单列排序和多列排序（列表传多列名，分别设升/降序）。筛选操作使用Pandas条件表达式（如df[df['col1']>2]实现数字筛选，df[df['col2']=='b']实现文本筛选），比Excel鼠标操作更灵活强大。查找替换在第9章详细讲解。本章展示了Python生态「组合使用」的核心思想——openpyxl处理Excel文件读写，Pandas处理数据分析逻辑。
