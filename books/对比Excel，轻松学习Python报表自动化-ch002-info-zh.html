02
基本概念
变量、数据类型、输出与缩进
Python中「数」分整数（int）和浮点数（float），「字符串」（str）用引号括起。变量是可以存放各类数据的容器，命名需以字母或下划线开头。print()函数用于输出，str.format()方法实现灵活格式化（.2f保留小数、.2%百分比）。Python用缩进识别代码块（通常4空格），#号表示单行注释。
📦

变量 = 容器

存放各种数据，通过变量名访问。命名规则：字母/下划线开头，区分大小写，不能是关键字。

🖨️

print() 输出

str.format() 实现一对一/多对多填充、浮点数精度控制(.2f)、百分数格式化(.2%)。

03
数据处理
字符串操作与三大数据结构
字符串支持九种操作：用+连接、用*复制、len()测长、in/find()查找、[]索引获取、[:]切片区间、split()分隔、strip()去首尾空格。列表（list）可变有序，支持append/insert/pop/remove/sort等增删改查。字典（dict）是键值对结构，用keys()/values()/items()遍历。元组（tuple）与列表类似但不可修改，两者可通过list()/tuple()互转。

04
控制流
循环、条件语句与函数
for循环遍历序列中每个元素执行操作，while在条件满足时持续循环（注意避免死循环）。if/elif/else构成条件判断链：先判断if，不满足则逐个判断elif，都不满足执行else。函数用def定义，可带参数和return返回值，通过函数名重复调用。匿名函数lambda更简洁：lambda arg: expression，适合简单的一行操作。
# for循环 — 遍历必修课 for course in ["Excel","SQL","Python","统计学"]: print(f"学习{course}") # 匿名函数 — 两数相乘 multiply = lambda x, y: x * y print(multiply(3, 4)) # 输出: 12 # 条件判断 if score >= 60: print("及格") elif score >= 80: print("优秀") else: print("不及格")
05
高级特性
列表生成式、map()与库管理
列表生成式用一行代码替代多层for循环：[x**2 for x in list]，数据量大时性能优势明显。map(function, args)对序列每个元素执行函数操作，结果可用list()转化。库是跨程序可调用的函数集合，pip install xxx安装、pip uninstall xxx卸载。本书主要使用openpyxl库，需在Anaconda Prompt（Windows）或终端（Mac）中安装。
