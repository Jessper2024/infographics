Review+ · Appendix A「Review Canvas」

A standardized review facilitation framework — a systematic discussion tool that progresses top-to-bottom, left-to-right

The Review Canvas is a standardized facilitation template based on the author's practice across multiple organizations, following the four-step method outlined in Chapter 2: Review Objectives → Evaluate Results → Analyze Causes → Summarize Insights. The canvas progresses from top-left to bottom-right: first fill in overview information and original intent/goals, then evaluate highlights and shortcomings against objectives, deeply analyze both objective and subjective causes, and finally formulate concrete action plans. The author emphasizes: the canvas is a facilitation tool, not a form-filling exercise — never reduce review to a mechanical checkbox process.

01
Canvas Structure
The Review Canvas: Top-Down, Left-to-Right Facilitation Table
The Review Canvas is a structured facilitation table — never reduce review to a form-filling exercise. The canvas works well for event/activity reviews; for major project reviews or strategic reviews, it serves only as a logical framework and meeting design reference, not for final presentation. Always conduct discussions top-to-bottom and left-to-right: first fill in overview information (topic, time, location, participants), then proceed through Steps 1–4 in sequence.

Step ①Review ObjectivesOriginal intent (purpose/driver) vs. concrete goals (SMART). If goals cannot be quantified, list key milestones.

Step ③Analyze CausesReasons for success/failure: both objective and subjective root causes. Use Fishbone Diagram, 5 Whys, or Thinking Compass (see Appendix B).
Step ④Summarize InsightsAction plan: concrete measures + timeline + responsible person. Start Doing · Stop Doing · Continue Doing. What to improve vs. what to sustain.
02
Step ①
Review Objectives: Distinguishing Intent from Goals
"Original intent" is the broader purpose and starting point; "goals" are the concrete, measurable indicators of whether that purpose was achieved — goals must follow the SMART rule. If quantifiable goals are difficult to set, at minimum list key milestones to measure progress and success. Clearly distinguishing intent from goals is the first step of review — many reviews fail because the initial goal-setting was vague.
🎯

Goals (SMART)

Concrete, measurable indicators. E.g., 20% sales growth, deliver by June 30. Provide objective benchmarks for evaluation.

💭

Intent (Purpose)

The fundamental reason and vision behind the action. E.g., improve customer satisfaction, establish industry benchmark. The intrinsic motivation.

03
Steps ②③④
Evaluate → Analyze → Summarize
When evaluating results, benchmark against objectives while also capturing process-level performance: identify both "highlights" and honestly confront "shortcomings." Cause analysis must cover both objective and subjective factors, digging deep for root causes — for complex causality, use Fishbone Diagrams, 5 Whys, or the Thinking Compass (see Appendix B). In the summarization phase, create specific, actionable plans — avoid vague platitudes.

🟢 Start Doing

New discoveries or remedial actions. Valuable practices not yet attempted but now needed.

🔴 Stop Doing

Measures that no longer work or have proven ineffective. What to abandon despite past investment.

🔵 Continue Doing

What has been well executed and deserves reinforcement. Proven effective practices to sustain and deepen.
04
Key Principle
Review Canvas ≠ Filling Out a Form
The core value of the Review Canvas is "guiding thinking," not "recording results." It is a tool for facilitators to drive deep team dialogue. Never reduce it to a mechanical checkbox task. The canvas's value emerges from the deep reflection, candid communication, and collective wisdom it stimulates. The correct way to use the canvas: let it spark dialogue and structure thinking, not merely document it.
05
Usage Scenarios
Using the Canvas Across Different Levels
The canvas is ideal for event/activity reviews; major projects and strategic reviews require more flexible application. For event-level reviews, the canvas can serve directly as a live facilitation and presentation tool. For major project reviews and strategic/business reviews, the canvas provides only the logical framework — actual execution demands more sophisticated meeting design, such as breakout groups, World Cafe, and multi-round iteration, with the canvas structure serving as the underlying logical backbone throughout.

Key Takeaway
The Review Canvas is a structured tool for guiding deep reflection — its value lies in sparking dialogue, not filling in boxes. The four-step method — Review Objectives (Intent + SMART Goals) → Evaluate Results (Highlights & Shortcomings, benchmarked against objectives) → Analyze Causes (Objective + Subjective, using Fishbone/5Whys/Thinking Compass) → Summarize Insights (Start Doing · Stop Doing · Continue Doing, with concrete measures + timeline + responsible person) — provides a clear logical path for every review. Remember: the canvas is for thinking, not for filling.
