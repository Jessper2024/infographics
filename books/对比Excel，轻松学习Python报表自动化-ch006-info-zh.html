对比Excel，轻松学习Python报表自动化 · 第6章「用Python设置Excel数字、条件格式」

SCQA · 情境 → 冲突 → 问题 → 回答 Excel的数字格式和条件格式是数据可视化的重要工具——将0.5显示为50%、将日期显示为年月日、将超标的单元格标红、给数据加上色阶或数据条。手工操作时，你要逐个选中区域、点击条件格式菜单、设置规则。但当报表每天更新且数据量庞大时，重复做这些事令人崩溃。如何用Python自动设置Excel的数字显示格式和条件格式？本章系统讲解number_format的9种数字格式、CellIsRule突出显示、DataBarRule数据条、ColorScaleRule色阶、IconSet图标集。

本章分为两大板块：数字格式和条件格式。数字格式部分，number_format属性对应Excel「数字」组，覆盖常规、小数（0.00）、百分比（0.00%）、货币（¥#,##0）、科学计数（0.00E+00）、日期时间（yyyy-mm-dd h:mm:ss）等9种常用格式。条件格式部分包括四种类型：CellIsRule()用于突出显示满足条件的单元格（支持greaterThan/lessThan/between/equal等7种operator）；DataBarRule()用于数据条可视化（可自定义min/max类型和值、颜色、是否显示数值）；ColorScaleRule()用于色阶（支持双色和三色刻度）；IconSet()用于图标集（多类图标可选、可设反转和是否显示数值）。两部分共同构成Excel数据可视化的Python自动化方案。

01
数字格式
number_format：9种数字显示格式一键切换
number_format属性直接映射Excel「数字」组。常用格式：General（常规）、0.00（保留2位小数）、0.00%（百分比）、¥#,##0（货币）、0.00E+00（科学计数）、yyyy-mm-dd h:mm:ss（长日期时间）、yyyy-mm-dd（短日期）、h:mm:ss（时间）。设置方式：cell.number_format = '0.00'，一行代码替代Excel的手动点击。

02
突出显示
CellIsRule：按条件自动标记异常数据
conditional_formatting + CellIsRule(operator, formula, fill) 实现标准条件格式。operator支持greaterThan、lessThan、between、equal、notEqual等7种。formula指定阈值（between用列表[50,80]）。fill使用PatternFill填充颜色。常用于标记超过预算的支出、低于目标的KPI、区间内的合格数据。
from openpyxl.formatting.rule import CellIsRule from openpyxl.styles import PatternFill # 大于50突出显示为红色 ws.conditional_formatting.add('A1:A10', CellIsRule(operator='greaterThan', formula=['50'], fill=PatternFill(fill_type='solid', fgColor='FF0000'))) # 介于50-80之间突出显示 ws.conditional_formatting.add('B1:B10', CellIsRule(operator='between', formula=['50','80'], fill=PatternFill(fill_type='solid', fgColor='FFFF00')))
03
数据条
DataBarRule：让数据大小一目了然
DataBarRule(start_type, start_value, end_type, end_value, color, showValue) 将数值以数据条形式展示。start_type/end_type可选min/max/num/percent/percentile/formula。color设置数据条颜色。showValue控制是否显示数字。同时作用于多列只需改区域范围如'A1:B10'。

IconSet：用图标直观展示数据等级
IconSet(iconSet, percent, cfvo, showValue, reverse) 将数据映射为图标。iconSet指定图标类型（3Arrows/3Flags/3TrafficLights等），cfvo列表定义各等级阈值，showValue控制是否显示数值，reverse控制是否反转图标方向。如同手机信号格，让数据等级一目了然。

🔑 核心结论
Excel的数字格式和条件格式是数据呈现的利器。Python中number_format属性覆盖常规、小数、百分比、货币、日期等9种常用数字格式。条件格式通过四种规则函数——CellIsRule（突出显示）、DataBarRule（数据条）、ColorScaleRule（色阶）、IconSet（图标集）——完整复现Excel条件格式的全部功能。它们的参数直接映射Excel对话框中的选项，学习路径清晰。批量应用只需将区域范围从'A1:A10'改为'A1:C20'，同一规则即可作用于多列数据。数字格式化与条件可视化全部自动化，报表的可读性和专业性大幅提升。
