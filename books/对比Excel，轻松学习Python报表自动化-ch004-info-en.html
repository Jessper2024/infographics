Compare with Excel, Easily Learn Python Report Automation · Ch.4 "Cell Selection & Font Settings with Python"

SCQA · Situation → Complication → Question → Answer Setting font formatting in Excel—font type, size, bold, color, background fill, borders—requires just a few clicks in the ribbon. But when you have hundreds of cells needing uniform formatting, or must repeat the same formatting daily for fixed ranges, manual operation becomes inefficient and error-prone. How can Python automate Excel cell formatting? This chapter systematically covers cell selection (single/row/col/range), font settings (Font() with 8 parameters), cell fill (PatternFill() solid and patterns), and border settings (Border() 4 sides + diagonals), culminating in a batch formatting case study that ties everything together.

This chapter focuses on the Python implementation of Excel's "Font" group in the "Home" tab. It first introduces multiple cell selection methods: direct coordinate (ws['A1']) or row/col numbering (ws.cell()) for single cells; row number, column letter, or range for multiple cells. It then dives into Font()'s 8 parameters: name, size, bold, italic, vertAlign, underline, strike, and color (ARGB format). Next it covers PatternFill() for cell fill—fill_type='solid' being the most common—and Border() for setting line style and color on all four sides and diagonals. Finally, a batch formatting case demonstrates iterating over multiple rows and columns to uniformly format all cells.

Selection
Cell Selection: Single, Rows, Columns & Ranges
Selecting cells is the prerequisite for formatting. Single cells: ws['A1'] (direct) or ws.cell(row=1,column=1) (numbered). Multiple rows: ws[5:10] slices; multiple columns: ws['C:E'] ranges; areas: ws['A1:C5']. These map directly to Excel mouse drag selections, making the learning curve gentle.

Font Settings
Font() Function: 8 Parameters for Complete Font Control
Font(name, size, bold, italic, vertAlign, underline, strike, color) is the core of openpyxl font formatting. color uses ARGB format (e.g., 'FFFF0000' for red). In practice, multiple settings are applied simultaneously: Font(name='Calibri', size=12, bold=True, color='FFFFFFFF'). Usage: cell.font = Font(param1=val1, param2=val2, ...).

Cell Fill
PatternFill(): Solid Colors & Pattern Fills
PatternFill(fill_type, start_color, end_color) controls cell background. Most common is solid fill: fill_type='solid', fgColor specifies the color. Pattern fills overlay a pattern on the background color (end_color). Usage: cell.fill = PatternFill(fill_type='solid', fgColor='FFFF00').

04
Border Settings
Border(): Four-Side Line Styles & Diagonals
Border(left, right, top, bottom, diagonal) configures cell borders. Each side uses Side(style, color) for line style (thin/double/dashed/etc.) and color. Diagonals require additional diagonalDown (top-left→bottom-right) and diagonalUp (bottom-left→top-right) parameters. Common practice: outer border = thin, header = medium, inner grid = hair.
from openpyxl.styles import Border, Side thin_border = Border( left=Side(style='thin', color='000000'), right=Side(style='thin', color='000000'), top=Side(style='thin', color='000000'), bottom=Side(style='thin', color='000000') ) cell.border = thin_border
05
Batch Practice
Iterative Batch Formatting: Apply Uniform Styles Once
In real scenarios, you rarely format a single cell. Use for loops to iterate over all cells in a range, uniformly applying font, fill, and border settings—this is the real value of report automation. Iterate by row (for row in ws[range]) or by column (for col in ws[range]), or limit to a specific area (e.g., B3:C5 only). Write once, reuse forever.

🔑 Key Takeaway
Excel formatting—fonts, fills, borders—maps to three core Python functions: Font(), PatternFill(), Border(). Their parameters directly mirror the Excel ribbon buttons, minimizing the learning curve. The real power is in batch operations: one for loop iterating over all cells, applying uniform formatting, making visual presentation as automated as data processing. This is the core value of Python report automation—eliminating all repetitive work.
