数据分析的定义与本质

数据分析是利用合适工具，在统计学理论支撑下，对数据进行预处理，结合业务分析数据，帮助业务部门监控、定位、分析、解决问题，从而高效决策、提高经营效率、发现业务机会点。

核心关键词：工具 + 统计学 + 预处理 + 业务分析 → 高效决策 + 竞争优势。数据分析的本质是将隐藏在大量数据背后的信息提炼出来，代替拍脑袋、靠经验的决策方式。

  • 啤酒与尿布：基于历史购买数据发现关联，实现搭配销售提高客单价
  • 谷歌流感趋势：通过关键词检索量异常预测流感爆发（思路值得借鉴）
2
三大分析作用

数据分析在企业日常经营中有三大核心作用：

  • 现状分析 — 告诉你业务过去发生了什么。通过KPI（订单数、新增用户、活跃率、留存率等）的同比/环比变化，了解整体运营趋势和各项业务构成。载体：日报、周报、月报。
  • 原因分析 — 告诉你某一现状为什么会存在。通过专题分析找原因：某商品销量为何突然大增？哪个区域表现突出？层层下钻找到问题根源。
  • 预测分析 — 告诉你未来可能发生什么。基于历史规律预测趋势，为制订季度/年度经营目标提供决策依据。例如：台风来临前多备面包。
3
六类分析指标

数据分析的重点在"分析"，不在工具。六个维度构成完整的指标体系：

  • 总体概览指标 — 统计绝对数，反映整体规模。如：日销售额60万、订单量2万、购买人数1.5万
  • 对比性指标 — 同比（同期对比）、环比（相邻对比）、差值。衡量变化方向和幅度
  • 集中趋势指标 — 一般水平。数值平均（普通/加权）和位置平均（中位数/众数）
  • 离散程度指标 — 数据波动。全距（极差）、四分位数、方差、标准差
  • 相关性指标 — 变量间关系，用相关系数 r ∈ [-1,1] 衡量。|r| 越大越相关，正负号代表方向
  • ⚠️ 相关关系 ≠ 因果关系 — 啤酒和尿布是相关，流感与关键词检索是因果，务必区分
4
九步常规流程

数据分析遵循"熟悉工具→明确目的→获取数据→熟悉数据→处理数据→分析数据→得出结论→验证结论→展示结论"九步标准流程。

  • ① 熟悉工具 — 工欲善其事，必先利其器
  • ② 明确目的 — 希望通过数据分析得出什么结论？
  • ③ 获取数据 — 明确指标、时间段、数据来源、提取方式
  • ④ 熟悉数据 — 数据量、数据类型、是否满足需求（与③双向循环）
  • ⑤ 处理数据 — 处理异常/重复/缺失/测试数据
  • ⑥ 分析数据 — 围绕指标，使用下钻法层层定位
  • ⑦ 得出结论 — 从分析中提炼结论
  • ⑧ 验证结论 — 与实际业务对照，用行动验证
  • ⑨ 展示结论 — 数据可视化呈现，分享给相关人员
5
工具对比：Excel vs Python

同一个操作可以用不同工具实现——Excel通过鼠标点选，Python通过代码。两者都是数据分析工具。

以"导入外部数据"为例：

  • Excel — 菜单栏：数据 → 根据文件格式选择对应选项 → 鼠标点选完成
  • Python — 编写代码：根据文件格式选择 read_excel() / read_csv() / read_table() 等函数

全书编排思路：针对数据分析的每一个操作，分别用Excel和Python对比实现，让读者在熟悉的Excel思维下平滑过渡到Python编程。

💡 核心结论

数据分析不是工具之争，而是方法论的掌握。理解"3大作用 + 6类指标 + 9步流程"这个框架，就掌握了数据分析的通用语言。Excel和Python只是这套框架的两种实施工具——前者通过鼠标操作降低门槛，后者通过代码编程提升效率和可复现性。全书后续章节将围绕"熟悉工具→获取数据→处理数据→分析数据→展示结论"这条主线，逐一进行Excel与Python的对比实现。
