两个中国：沿海繁荣与内陆困局的断裂

全球视野
世界不是平的：全球经济活动高度集聚
世界银行《2009年世界发展报告》指出，全球半数经济活动位于仅1.5%的陆地区域。 从夜晚地球灯光图可见，经济发达的地方灯光密集——这并非偶然，而是经济规律使然。越发达的国家，经济集聚程度越高。美国的GDP高度集中在东海岸、西海岸和南部少数地区，日本三大都市圈集中了50%以上的人口。中国的人均GDP和人均收入差距恰恰是因为人口集聚还不够，而非过度。

第2章 · 中国经济的欧洲化

本章从欧债危机切入，揭示一个被忽略的警示：中国最需要担心的不是"拉美化"，而是"欧洲化"——确切地说，是"欧元区化"。欧元区的困境源于市场分割与统一货币的冲突：由于人口未能充分流动，地区间劳动生产率难以趋同；当弱国丧失独立的货币政策和汇率工具时，债务危机便成为必然。将这一逻辑映射到中国省际层面——如果人口流动受阻，地区间收入差距长期存在，统一财政和货币政策的中国将面临与欧元区相似的结构性困境。本章进而深入分析大国规模经济的好处，以及统一市场面临的内生挑战。

类比中国
从欧元区到中国省际：同一个逻辑的不同尺度
对于欧洲是国家间的问题，放在中国就是省和省之间的问题。 欧元区的困境对中国的启示是：在货币统一的区域内，人口自由流动是必需的。否则，地区间劳动生产率趋同难以实现。欠发达地区因为劳动生产率低、人口无法流出、福利支出刚性、工资调整困难，只能通过借债弥补。中国各省之间虽然不存在汇率问题，但面临同样的结构性矛盾：人口流动不充分 → 区域差距持续 → 行政干预扭曲资源配置 → 效率损失。
🇪🇺

欧元区的教训

多国、多语言、多文化 → 人口流动不充分 → 劳动生产率差异持续 → 丧失汇率工具 → 债务危机。各国的局部理性导致全局困境。

🇨🇳

中国的映射

省际户籍壁垒 → 劳动力流动受阻 → 区域人均GDP差异巨大（贵州人均GDP仅为京津沪的1/4）→ 行政配置资源替代市场力量 → 效率与平衡的冲突加剧。

04
治理挑战
大国的治理难题：统一与分权的千年博弈
大国并非越大越好，规模带来的收益必须与治理成本相权衡。 秦始皇统一后做了两件大事："车同轨，书同文"——统一文字使今天的北京话和闽南话仍是方言而非两种语言；建设驰道和驿站以维护统一。此后两千多年，历代王朝都在集权与分权之间摇摆：分封制下地方诸侯威胁统一，郡县制则把地方官任免权收归中央。今天，省级官员仍由中央任免。但治理一个14亿人口、50多个民族、差异巨大的大国，成本仍然极高——民族间交易成本堪比国家间贸易成本。
👑

集权模式（郡县制）

地方官由中央任免，权力集中，有利于统一。但中央与地方信息不对称，容易产生"一刀切"的政策失误。

🔱

分权模式（分封制）

地方有较大自主权，适应性强。但诸侯势大则威胁统一，历史教训表明分权过度最终导致王朝覆灭。

🔑 核心结论
中国真正应该警惕的不是"拉美化"，而是"欧洲化"（欧元区化）。欧元区的困境源于市场分割与统一货币的内在冲突——人口不能充分流动时，统一货币就会成为灾难的放大器。中国作为比整个欧元区还要大的统一经济体，面临同样的结构性挑战。破解之道不是限制人口流动、用行政力量配置资源，而是消除流动障碍，让市场力量引导人口向高生产率地区集聚，在流动中实现人均收入的均等化。大国规模经济的好处巨大，但须以要素自由流动为前提。

第3章 · 在集聚中走向平衡

本章回应关于空间集聚的三个流行误解：中国人口过度向少数地区集聚了吗？没有政府管制人口会无限集聚吗？不会。人口集聚会导致地区差距越大越大？恰恰相反。作者用跨国比较数据和经济发展规律证明：越发达的国家集聚程度越高；集聚程度在发展到一定阶段后趋于稳定（不会无限增长）；集聚最终会走向人均收入意义上的平衡——只要劳动力可以自由流动。全章的核心是区域经济学的精髓：在集聚中走向平衡。

误解二
"人口会无限集聚？"——集聚有自然边界，由市场调节
集聚不会无限持续，因为生产要素价格会自动调节。 在发达国家发展过程中，集聚程度不断上升，但到一定阶段就稳定了——不是因为政府管制，而是因为中心区的地价、房价和工资太高，使生产者和消费者自行重新评估"是否值得留在大城市"。东京就是一个典型案例：日本人口总数在下降，但东京都和东京圈的人口仍在增长——这就是集聚的力量。市场的价格机制是更有效的"人口调控者"，远比行政限令明智。
📈

集聚的动态过程

经济快速发展 → 人口快速集聚到中心区 → 地价、房价、工资上涨 → 部分人口/企业迁出 → 集聚速度放缓但程度不降 → 东京人口持续增长即为例证。

🏙️

市场调节机制

集聚好处（高收入、多样化消费、知识溢出） vs 坏处（高房价、拥挤、污染）共同决定均衡。要素价格（地价、房价、工资）是核心调节变量。

03

自由移民
自由移民的意义远超权利保障——它是区域平衡的引擎
自由移民不仅是人民的基本权利，更是缩小城乡和区域差距的根本机制。 回到松茸的故事：为什么卓玛不能在城里的餐馆烤松茸？户籍、土地、公共服务等制度壁垒是显性障碍。此外，少数民族进城还面临语言和文化障碍。自由移民的机制是双重的：一方面，进城的"卓玛"们收入成倍上升；另一方面，留在老家的"卓玛"们因为人少了，人均收获大幅提高。最终，只有当城乡间和地区间的实际收入差距缩小至零（扣除物价和心理成本后），人口才会稳定下来——这才是真正的平衡。

关于空间集聚的三个流行误解全部不成立：（1）中国的集聚程度在国际上不仅不高，反而偏低；（2）集聚不会无限持续，要素价格的自发调节是天然的稳定机制；（3）集聚不是扩大区域差距的原因——相反，劳动力的自由流动才是实现人均意义上区域平衡的唯一途径。区域经济学的精髓只有一句话："在集聚中走向平衡"——这不是口号，而是被全球经验反复验证的经济规律。中国户籍制度和土地政策对人口流动的限制，恰恰是阻碍区域平衡发展的最大障碍。

政府与市场
城市化的"立法"：政府与市场的边界在哪里？
市场的力量：城市扩张的收益（高收入、消费多样性、知识溢出）与成本（拥挤、污染、高房价）由个体和企业权衡。利润和工资反映生产率，引导资源最优配置。政府的作用：纠正市场"外部性"——通过基础设施投资放大正外部性（如地铁缓解拥堵），通过技术和管理减少负外部性（如治安摄像降低犯罪率）。政府改善"城市病"的后果不是减少城市人口，而是增加城市人口。没有任何理论依据支持政府直接采取行政手段控制人口，更没有理论支持将城市人口划分为"需要的"和"不需要的"。
📊

市场做的事

决定人口和企业的最佳选址：利润和工资作为价格信号，引导资源配置。地价和房价自然成为低效率企业的进入门槛。

🏛️

政府做的事

纠正外部性：增公交减拥堵（正外部性）；装摄像头降犯罪（负外部性）。这些干预会增加而非减少城市人口容量。

破解之道
破解"不可能三角"：重新定义平衡，释放市场力量
破解"不可能三角"的关键在于放弃"均匀分布"意义上的平衡，追求"人均"意义上的平衡。 这不是文字游戏，而是根本性的政策理念转变。从欧洲到美国，从日本到中国，经济规律从未改变：人口流向高生产率的地区，在流动中实现人均收入的均等化。政府不应限制大城市发展，而应致力于：完善基础设施和公共服务，纠正城市发展中的负外部性；对于欠发达地区，提供必要的公共服务转移支付（而非引导生产性投资）；消除户籍等制度障碍，保障劳动力自由流动的权利——这既是基本人权，也是经济规律的要求。

追求资源均匀分布 → 行政配置资源 → 效率损失、市场分割、人均平衡无法实现 → 双输

新思路（出路）

追求人均平衡 → 消除流动障碍 → 市场引导人口流向高效率地区 → 人均收入趋同 + 国家效率提升 → 双赢

🔑 核心结论
大国发展存在一个"不可能三角"：统一、效率和资源均匀分布无法同时实现。中国过去十几年走的正是这条死胡同——用行政力量向中西部倾斜资源，限制东部大城市发展。结果是：TFP增长率持续下滑、企业间效率差距扩大、产业被迫外迁至东南亚、中西部工业园大量空置。破解之道在于重新定义"平衡"——从"资源均匀分布"转向"人均GDP和生活质量意义上的平衡"。这要求政府停止用行政手段干预资源空间配置，转而消除人口流动的制度障碍，让市场力量引导资源流向效率最高的地方。唯有如此，统一的大国才能同时实现效率和平衡。

前现代大国
古代皇帝的三件大事：收税、分散风险、国防
古代的中国分分合合，最终走向统一，不是偶然。古代皇帝想三件事：第一是收税——"普天之下，莫非王土；率土之滨，莫非王臣"，国家当然越大越好。第二是分散风险——同样一场地震，小国可能毁灭，但在中国这样的大国，"一方有难八方支援"。修建连通首都和其他地方的道路和运河，很重要的功能就是保证皇城粮食供给不受地方灾害影响。第三是安全——中原地带以平原为主，一旦分成若干小国则战火不断，统一则长治久安。大国在国防上还有规模经济效应——人均军费可以被摊薄。

最优规模
大国不是越大越好：规模收益与治理成本的权衡
作者明确指出：尽管大国有很多好处，但并非越大越好。随着国土面积增加，人口规模、民族多样性和自然资源也会增加，治理成本随之上升。一个国家的"最优规模"取决于规模收益与治理成本之间的权衡。 不同民族之间的交易成本堪比国家间贸易。即便在"二战"后没有大规模战争的和平时期，许多国家也因这种权衡而调整了规模——最著名的是"冷战"后苏联的解体。对于中国来说，关键不是争论规模本身，而是如何在现有规模下，通过降低治理成本（打破市场分割、改革户籍制度）来最大化规模收益。
08
治理历史
集权与分权的千年摇摆
中国古代治理的核心难题是集权还是分权。秦始皇建立郡县制，把地方官的任免权收归中央。但国家太大，完全集权管不过来，于是出现了分权的做法——西汉把全国分封给皇亲国戚。到了现代，毛泽东在《论十大关系》中强调要发挥中央和地方两个积极性。改革开放后的分权体制一方面激励了地方政府发展经济，但另一方面也埋下了市场分割的隐患。 每个地方都想保护本地产业，导致重复建设、产业同构、地方保护主义，这些因素共同抑制着中国发挥大国应有的规模经济。

制度红利
从人口红利到制度红利：中国的十字路口
随着人口红利基本结束，劳动和资本两大生产要素增长放缓，中国经济可能进入增长速度下滑时期。未来20年，如果中国能持续中高速增长，将跨越"中等收入陷阱"。增长的动能源自何处？答案是"转型与发展中的大国"。 在"发展"意义上，最重要的是城市化——目前城市化水平仅过半，要达到75%的覆盖率仍需20年。在"大国"意义上，不同地区、不同城市具有不同的条件，如果生产要素能充分自由地跨地区配置，将形成差异化的分工体系，进一步提高资源配置效率。制度上的"转型"——从人口红利转向制度红利——就是改善资源配置效率的战略路径。
🔑 核心结论
大国之所以为"大"，不只是国土面积和人口数量，更在于规模经济带来的独特优势：从战略性产业（大飞机）到技术创新（淘宝、微信），从公共品供给（北斗导航）到国际货币（人民币国际化），再到现代服务业（金融、教育、文化）的竞争力——"大"本身就是一种经济资源。但大国治理也面临深刻挑战：政治集权与经济分权之间的张力导致市场分割和要素流动障碍；人口红利行将耗尽，制度红利却还未充分释放。站在十字路口的中国，必须认识到：大国的优势不是自动得来的——它依赖于统一大市场的建设。如果市场被分割，大国将变成"大而不强"的国家，甚至走向"欧洲化"的危险境地。从"大国体量"到"大国优势"，中间的关键一步就是打破壁垒、促进流动、统一市场。

欧债危机
希腊的困境：市场分割+统一货币的两大根源
欧洲债务危机的深层根源是两大因素：第一，市场分割——尽管欧盟没有人口流动的制度障碍，但语言、文字、宗教、文化差异导致欧洲内部人口流动不充分、不自由。法国、德国人均GDP大约是希腊的两倍。第二，统一货币——无法独立的货币政策使希腊在经济衰退时无法通过本币贬值刺激出口。当工资无法灵活调整，汇率又由欧洲平均劳动生产率决定时，希腊的汇率就"高估"了，出口难以提升，经济增长受阻，只能靠举债维持社会保障。这与中国省际问题高度相似：中国虽有统一货币，但劳动力流动的制度障碍甚至比欧盟国家间还大。中国会不会走向"欧洲化"？
02
经济逻辑
欧元区的根本缺陷：劳动生产率趋同无法实现
欧元区的深层矛盾在于：当结成货币联盟的国家存在巨大发展差距时，统一汇率的负面效果便在危机中尽显。 欧元汇率由各国的平均劳动生产率决定——对德国、法国偏低，对希腊等"边缘国家"偏高。偏高的汇率使出口受限，经济受制约。而人口又未充分流动起来，国家却要为自己的国民提供刚性的社会保障和公共服务，唯一的出路就是借债。危机前欧洲货币政策偏松，弱国借机大肆举债。危机来临时，虚假繁荣的经济显得异常脆弱。如果有自己独立的货币，希腊就应该贬值；但在欧元区，各国已经丧失了独立的货币政策。对于中国，统一的货币政策下，中西部欠发达省份面临的困境与此极为相似。
03
中国镜像
中国的地方政府债务：一个潜在的"危机"

道德风险
统一财政的"优势"与陷阱
很多人认为中国有统一财政的优势可以避免希腊式的主权债务危机。但作者指出：恰恰是这种预期——中央会救助负债累累的地方——带来了严重的"道德风险"问题。 地方政府明知偿债困难，却因为预期中央兜底而继续举债。如果说地方政府债务主要投入了经济建设，那应该是有产出的。问题是，大量超标建设的新城和工业园永远不会有回报了。更严重的是，如果无法通过经济增长提高分母（GDP），就只能通过通货膨胀来减轻债务负担——而通胀本质上是对全民征税。因此，不要认为表面上没有出现债务违约就安全了。地方政府无法自行偿还的债务，最终都要全民一起还，不是通过税收就是通过通胀。

两难选择
救希腊还是让希腊出局？大国责任与选择
2008年全球经济危机后，希腊成为第一个主权债务违约的发达国家。欧洲面临艰难抉择：救希腊，还是让其退出欧元区？救希腊意味着用西欧的钱补贴"边缘国家"；不救则意味着一体化进程的倒退。作者指出：欧元区的尴尬带来了一个对中国的重要启示。 在货币一体化的区域（或国家），人口自由流动是必需的——否则地区间劳动生产率趋同难以实现。欠发达地区劳动生产率较低，如果人又没有流动起来，社会福利和公共服务的支出又是刚性的，靠举债弥补收支差额就在所难免。中国各省之间关系的核心问题在于：各个省如果把彼此的关系当成"国家间"关系，中国作为一个大国的公共利益将严重受损。
06
囚徒困境
"不明白"和"装糊涂"：政策僵局的心理根源
为什么打破城乡间和地区间要素流动障碍迟迟未能实现？作者给出两个答案：第一，在知识层面，社会各界还没有充分认识到生产要素跨地区流动的必要性，城市发展和经济集聚的坏处被高估了。 这就是"不明白"——知识层面的不足。第二，即使知识问题解决了，各个地区都陷入了一个以本地和短期利益高于全局和长期利益的"囚徒困境"中——"就算大家都认识到市场整合是有利的，但每个地区都仍有动机去分割市场。" 这就是"装糊涂"。最重要的表现：每个地方都想要更多建设用地指标、更多转移支付、更多高技能劳动力，而对于已经流动到本地的低技能劳动力却不愿意接纳为永久市民。
07
国际比较
与欧洲比：中国户籍制度障碍超过欧盟国家间壁垒
与欧洲相比，中国本应更好地发挥大国优势——统一的货币和共同市场。但实际上，中国的劳动力流动制度障碍比欧盟国家间还大。 在很多国家（不仅仅发达国家），长期居住的外国人可以享受基本公共服务（医疗、教育、低收入补贴）。而在中国，没有本地户籍的常住人口却无法均等化享受公共服务。大城市对外来人口获得本地户籍设置的门槛堪比欧美国家对外国移民获得居留权。这是非常荒谬的——中国省与省之间的人口流动障碍居然比国家间的障碍还要大！欧盟在努力推进一体化，而中国这个统一国家却在制度上阻碍劳动力流动，这种矛盾不可持续。未来不推进市场整合，中国经济将面临"欧洲化"的危险。

转型与发展中大国：城市化释放制度红利
中国与1980年代日本有表面相似（贸易顺差、本币升值、房价上涨），但本质不同：中国是一个"转型与发展中的大国"，这是最大优势。 30年前的日本已基本完成城市化和工业化，服务业占GDP超过63.2%。而中国的城市化进程才刚刚过半，如果不受抑制，还将为经济增长释放新的活力。在"发展"意义上，城市化推进过程中，城市有利于人口集聚、知识溢出和人力资本提升，进而持续提高劳动生产率。在"大国"意义上，不同地区、不同城市有不同禀赋，要素自由流动可形成差异化分工体系。低技能劳动力的市民化不是负担，而是城市长期竞争力提升的基础。但制度障碍恰恰阻碍了这一进程。

政策路径
自由移民：不只是公平，更是国家发展战略
作者强调：自由移民不只是公平与否的问题，更是"国家发展战略"。 如果中国想长期发挥大国优势、避免欧洲的困境，就必须实现区域之间人均GDP的趋同，而最优路径就是国家内部的自由移民。这一结论基于一个核心经济逻辑：给定统一货币和统一汇率，工资虽有刚性但最终会反映劳动生产率差异；如果劳动力自由流动，地区间的劳动生产率和人均收入就会趋于均等化。否则，只能通过维持地区间巨大收入差距来补偿——而这在任何国家都不可能成为一个政策目标。因此，劳动力自由流动，最终实现地区间的劳动生产率和收入均等，是大国发展唯一可行的战略选择。

平衡观
"在集聚中走向平衡"：人均意义上的平衡
区域经济学的精髓就是一句话——"在集聚中走向平衡"。 人们通常混淆两个概念：总量的差距 vs 人均的差距。 把全国每个城市都建成上海一样大，这违反经济规律。真正有意义的是地区间实际收入和生活质量的差距最小化。世界银行的研究结论：所有实现了地区间平衡发展的国家，都是劳动力自由流动的国家；凡是通过行政配置资源追求地区平衡的，没有成功的。生产要素流动使要素回报均等化——这是经济学的基本原理。 美国和中国的GDP份额、人口份额对比图给出了最直观的证据。

自由移民
自由移民：个人权利与国家战略
自由移民的双重意义：第一，保障个人权利。 人们有权到能够获得更高收入的地方去工作。"卓玛与松茸"的故事说明——如果不变革制度不让人进城，采松茸的人太多，人均收入永远无法提高。第二，缩小城乡和地区间收入差距。 从长期来看，只有当城乡和地区间的实际收入差距缩小至零时，人口规模才会稳定下来。劳动力自由流动后，各地形成分工与合作——有的地方造汽车，有的搞旅游，有的采松茸。地区间形成经济依赖，是保持国家统一和市场整合的最有效途径。"用脚投票"机制也倒逼地方政府提高公共服务质量。

不可能三角
大国发展的"不可能三角"：统一、效率与均匀分布
如果同时追求三个目标——国家统一、经济效率、区域间均衡发展——而将"平衡"理解为经济资源和人口的均匀分布，并通过行政手段实现，那么这三个目标不可能同时实现。这就是大国发展的"不可能三角"。破解之道：重新定义"平衡"——从资源和人口的均匀分布，转变为人均GDP、人均实际收入和生活质量意义上的平衡。这种人均意义上的平衡恰恰可以在人口自由流动中实现。反之，如果坚持均匀分布，就会失去经济效率，也会失去市场统一——每个地方政府都想做大自己，在缺乏竞争力时分割市场。
02
政府与市场
政府与市场的边界：为城市化"立法"
城市化争论的核心不在速度和水平，而在于"在哪里城市化"。市场的作用：让企业和劳动者在收益与成本之间自行权衡。 大城市的收入高、消费品多样性多——但也需面对高房价、长通勤、拥堵和污染。这些问题本身就是地区间生活质量平衡的市场机制。政府的作用：弥补市场缺陷。 通过基础设施和人力资本投入放大"正外部性"（知识溢出、相互学习），通过技术和管理减少"负外部性"（拥堵、污染、犯罪）。关键结论：并没有理论支持政府直接控制人口，更无理论支持将一个城市的工作人口区分为"需要的"和"不需要的"。

产业转移
"腾笼换鸟"的悖论：鸟飞到了东南亚
通过行政力量推动产业转移，结果往往事与愿违。企业家的目的是挣钱，不是扶贫。与中国中西部相比，东南亚国家工资更低，深水港条件不差，离欧洲更近。 所以大量劳动密集型企业没有转移到中西部，而是直接转移到越南、柬埔寨等国。2015年以来，这种产业外迁趋势越来越明显——看看出国时国外卖的服装玩具产自哪里就知道了。而转移到中西部的企业不少是国有企业、资源型企业和污染企业。更让人担心的是：中西部和东部的产业结构出现趋同趋势。 有研究发现，中国工业地理集中和行业专业化的程度在2005年前后由上升转为下降，省际产业同构现象加剧——与区域间分工协作背道而驰。

区域差异
城投债、负债与不可持续的"增长"
中西部城市建设往往依赖政府负债，而工业园和新城的投资效率令人质疑。2011年中西部城投债增量份额约50%，而其GDP份额仅约41%。这些债务投入了大量超标建设的新城和工业园，而这些地方并没有足够的企业和人口进入。在短期内，投资的拉动表面上促进了中西部经济增长，但这种增长缺乏效率支撑，难以持续。更严重的是：偏离当地比较优势的投资不仅不会有效帮助欠发达地区发展，反而带来沉重的财政负担。 作者明确指出：这些投资不只是"没有回报"——有些"永远都不会有回报了"。如果地方政府欠债不还，而居民又不能自由流动出去，这个地方就成了"孤岛"。人均意义上的平衡才是真正有效的平衡。

破解之道
破解"不可能三角"：重新定义平衡
大国发展的"不可能三角"并非无解。破解的关键在于重新定义"平衡"——放弃追求资源和人口的均匀分布，转而追求人均GDP、人均实际收入和生活质量的趋同。这种人均意义上的平衡可以在人口自由流动过程中实现，不需要采取大规模行政控制手段。正如世界银行的研究：所有实现了地区间平衡发展的国家，都是劳动力自由流动的国家。反之，追求均匀分布，就必然会失去效率，也会失去市场统一。这本质上就是在谈政府和市场的关系——将这对关系放到城市化和区域经济发展的具体问题里去。

🔑 核心结论
大国发展面临一个"不可能三角"：统一、效率和均匀分布意义上的平衡三者无法同时实现。破解之道是放弃"均匀分布"的思维定式，转而追求人均意义上的平衡。中国的现实问题在于：行政配置资源的方向与人口流动方向完全相反——人往高处走，资源向低处流。2003年以来，这种违背经济规律的政策已导致TFP增长率下滑、资源配置效率恶化、地的城市化快于人的城市化、中西部产业同构和债务负担加重。世界各国的空间经济图谱清晰地表明：世界不是平的，经济集聚集聚是不可抗拒的内在规律。正确的政策方向是尊重市场规律，让"动人"（劳动力自由流动）优先于"动钱"（财政转移），在集聚中实现人均意义上的平衡。

第11章 · 警惕'扭曲之手'

01
公平原则
区域发展的公平三原则
高铁应该连接大城市还是每个小城市？作者与律师朋友的对话引出了区域发展公平性的核心框架——"公平三原则"： 第一，公共投资的效率原则——高铁连接大城市是有效率的，不应让高铁经过每一个县和村。第二，人口的自由流动——小城市的人口可以通过自由迁徙获得高铁的好处。第三，财政转移支付——没有直接享受到高铁好处的地方，应通过财政转移分享国家发展成果。三个构件缺一不可。这是理解政府和市场关系的核心框架："政府更好地发挥自身功能，而不是取代市场的功能。" 财政转移应主要用于公共服务（教育、医疗），而非生产性投资——因为人力资本可以随人流动，而固定投资一旦建成却无竞争力，就会成为"鸡肋"。
02
联动改革
土地和户籍制度的联动改革
中国的建设用地指标配置制度严重制约了区域经济格局的调整。人口流入多的城市用地指标紧张，人口流出的中西部却有指标没企业。 解决方案：让沿海城市购买中西部的建设用地指标。 进城农民老家的宅基地复耕后，对应的指标可以有偿转让给工作的城市。这样中西部保留更多耕地，沿海发展工业，劳动力向东部转移，和户籍一起联动改革。土地利用效率的巨大空间： 从大港口向内陆走500公里（约上海到安徽西部），土地利用效率下降50%。距离港口450公里以内，城市扩张的规模效应占主导；超过450公里，负效应占主导。这意味着在未来城市化中，让距港口450公里以内的城市扩张更快，才有利于提高土地利用效率。
03
正义论
"无知之幕"与公共政策的公正性
作者引入罗尔斯的"无知之幕"理论：公共政策讨论中，每个人的观点应与社会身份无关。 一个公正的政策应该达到这样的状态——不管你是政策的直接受益者还是受损者，如果把位置调换，这个政策仍然会被认为是合理的。当前很多政策"不过关"：大城市欢迎外来人口打工，但"孩子和老人不要来"。反过来想：如果你自己是打工者，你愿意自己在城里打工，把孩子和老人留在老家吗？"当今中国的城市人往上数三代，大多都是从农村进城的。" 中国很快就会成为全世界经济体量最大的国家，但"发达是物质层面的，价值观才是文明层面的"。如果不能平等对待自己的国民，将来如何让别的国家相信中国会平等对待它们？
04
地理效率
到大港口的距离决定土地利用效率

05
劳动力
劳动力都去哪儿了？——"短缺"的真相
一方面城市出现招工难，一方面农村出现空心村——这真的说明中国劳动力严重短缺了吗？2014年底城市化率只有54%——刚过半。中国的劳动力流动中，举家迁移率只有20%左右。 在不能举家迁移的情况下，当然是年轻男性进城打工。结果是：5000万留守老人、1000多万留守女性、6100万留守儿童。 农村的老龄化和女性化不是劳动力枯竭的信号，而是制度约束的结果。如果改革到位，大量家庭仅输送年轻男性的局面就会改变。服务业的比重将越来越高，而服务业对体力要求远低于制造业——未来农村居民举家迁移的趋势将越来越强。我们不应该担心的不是劳动力总量枯竭，而是城市是否具有不断提高劳动生产率的能力。

06
农业转型
人口流出与农业转型：农场规模与粮食价格
很多人担心农村人口减少会威胁农业生产。但作者指出：农业产出主要取决于土地面积，而不是人口数量。 过去中国城市化推进的同时粮食产量在增长。随着农村人口减少，机器正在替代劳动——这就是劳动生产率的提高。中国的农业人均土地面积只有美国的1/400，欧盟的1/40。人均土地面积小必然导致农业收入难以提高。持续减少农业人口才是提高农产品价格竞争力的出路。 一个耐人寻味的现象：中国粮食库存上升的同时进口也增长迅速——因为政府保护价收购使国内粮价高于进口价。美国粮食价格低，原因就是规模经营。从长期看，随着农场规模扩大，农村人口结构会"内生"调整——年轻男性和较高教育水平的劳动力会回流到农业中。人口流出是农村地区提高规模经营程度和劳动生产率的前提条件。
07
流出视角
人口流出未必不好：换个角度看空心村
中国农村大量"空心村"的出现让人忧虑，但换个角度看：人口从欠发达地区流出是资源优化配置的结果。 发达国家走过的城市化道路中，也一定不断出现空心村。即使在城市化完成之后，随着人口向少数大都市圈集中，仍然有不少城市的人口在萎缩——底特律人口从1950年180万降至2010年的71万。挪威2005—2009年的人口流动图中，大城市都是深红色（净流入），小城市大多是蓝色（人口流出）。人口流出本身并不是可怕的事情。 美国80%的人集中在4%的国土上，但各州人均GDP高度一致。中国是正处在城市化中的国家，空心村和人口流出的出现恰恰是人口在空间上优化配置的结果。未来，一些古村落逐步演变为旅游地，宅基地复耕为农业用地——到底哪个村落有保护价值，由市场来决定。
08
土地权利
农村土地改革：不能以禁止交易来"保护"农民
反对农村土地改革的人有一个荒谬的逻辑：以禁止市场交易的方式来保障农民。 作者指出，土地如果不能交易，本质上就不是资产。关键不是禁止交易，而是确保自愿、有就业、有社保、市场定价这四大前提。作者在重庆贫困农村调研时，一个农民反问："城里人可以用自己的房子抵押贷款，为什么农民的土地不可以？" 另一个反对观点认为土地流转会导致"土地兼并"——但中国早已不是农业经济，农业占GDP已经降到10%。城市不断创造新收入和就业，农民对家乡土地的使用权如何处置，应让其自由选择。"为了担心土地价值被低估，就不让农民做交易"——这是最荒谬的逻辑。 禁止宅基地交易的结果是什么？大量农村住房已经"人去楼空"，有些还是新盖的——因为人们认为最后不得不回到农村，这些投资很多是浪费。
09
东北案例
东北经济告急？人均GDP高速增长的真相
媒体上很多文章说"东北经济告急"——人口大量净流出、老龄化严重。但作者指出：人口流出不意味着经济衰退。 2014年东北三省人均GDP是10年前的约3倍。扣除物价后，辽宁和吉林的人均GDP增长速度都超过了全国平均。辽宁2014年人均GDP是2005年的2.6倍，吉林是2.8倍，黑龙江是2倍，全国是2.5倍——辽宁和吉林跑赢了全国。"我没看出有什么'告急'的情况嘛。" 关键是看人均GDP。当人口流出时，即使GDP总量不变或缓慢增长，人均GDP也是快速增长的。对于那些缺乏发展工业和服务业优势的地区来说，人口流出恰恰是提高居民人均收入的有效途径。与其盲目建设基础设施和住房来"吸引"人口，不如利用当地比较优势发展旅游和大规模农业——正是因为有较少的人口，这些产业才能为留下来的居民提供更高的收入。
10
改革方向
"动人"与"动钱"的良性互动
通过"动人"（劳动力自由流动）推动经济发展和实现区域平衡是长久之道，但这并不是说"动钱"（财政转移）的事不要做。恰恰相反，在"蛋糕做大"的情况下，中央财政更有能力进行区域间和城乡间的财政转移。未来的财政转移应更多用于城乡和区域间公共服务的均等化——提高欠发达地区中小学教师和医护人员的待遇，减少欠发达地区政府的财政负担。教育医疗投资形成的人力资本是可以随人流动的， 而生产性投资一旦成了固定投入却没有竞争力，就会成为"鸡肋"。作者总结：中国当前的问题不是城市化过度，而是城市化进程受阻。 城市排外政策让家庭分居、人力资本浪费。特大城市把减少外来人口子女受教育机会当做政绩，这会让这些城市成为"全人类的例外"。改革的方向很清楚：消除制度障碍，让人和资源在空间上自由流动和优化配置。

🔑 核心结论
区域发展的"公平三原则"——公共投资效率、人口自由流动、财政转移支付——三者缺一不可。土地和户籍制度的联动改革是释放中国下一轮经济增长动力的关键。到大港口距离对土地利用效率的决定性影响说明：地理对于经济发展是重要的，不能用行政力量去逆转。中国的问题不是城市化过度，而是城市化进程受阻——举家迁移率仅20%，6100万留守儿童是人类制度扭曲的结果。人口流出不可怕，东北三省人均GDP增长速度跑赢了全国就是明证。未来改革的方向是消除制度障碍，让"动人"（劳动力自由流动）优先于"动钱"（财政转移），实现区域间的公共服务均等化，让每一个中国人在这个统一大国内自由选择自己的生活和就业地。

第13章 · 大城市不死

本章是全书的第5章（书第五章）。核心论点：大城市不会"死"，规模经济效应、人力资本外部性和技能互补性使大城市在现代化经济中不可替代。作者从"80后"朋友小陈博士夫妇选择留在北京的个人案例入手，阐明大城市发展的两个关键逻辑：人力资本积累和专业化就业匹配。本章先论大城市的生机源自人性——喜欢多样性和交往。再以实证数据证明，城市规模扩张能创造包容性就业，尤其有利于低技能者（城市人口每增加100万人，个人就业概率提高0.66个百分点）。进而比较"生活在城市"和"活在城市"两种发展模式——窄马路高密度vs宽马路低密度，用上海浦西田子坊和浦东联洋社区的对比说明密度对服务业发展的重要性。最后批判"以业控人"的思维方式，揭示高技能者与低技能者的互补关系：一个高科技岗位带动5个其他行业的就业。

01
城市生机
大城市的生机：人性、交往与多样性
世界上有两种人：一种向往陶渊明"采菊东篱下"的离群索居，另一种喜欢热闹和多样性。绝大多数人属于后者。现代经济中，知识密集型产业比重越来越高——越来越多的人需要通过和别人的交往来传播知识、获取知识。 从前再喜欢热闹的农民也必须在农村和土地为伴；现在再内向的白领也必须在最低限度上与人交流。人的另一天性是喜欢多样性——只有大城市才能提供丰富多彩的消费服务。在大城市，生产者可以针对某一细分市场提供服务，因为每一种特殊偏好的消费者在庞大的城市里都足够多。高房价和"城市病"就是消费者为获得大城市多样性生活所支付的"入场券"。 如今，"消费型城市"越来越重要，一个城市的生活品质正在成为人口迁移的主要吸引力。
02
信息技术
线上交流不会取代线下——互补而非互替
有人担心信息技术会降低人们集聚在城市的必要性。但事实恰恰相反：线上和线下的交流是"互补"而非"互替"。 微信朋友圈里都是见过面的朋友；刚开始在线上交流的人，接下来会面对面讨论。信息技术只是节省了与某个特定朋友见面的次数，却可能催生了与更多人交流的需要。人越有钱越会购买"体验"——吃饭穿衣的需求会饱和，但音乐会、戏剧、画展、咖啡馆的"体验"消费是不断提高的。如果线上可以取代线下，就不会有那么多人愿意花很多钱、拖着疲惫的身躯到商学院课堂去上课，而不是在网上看教授的视频。 日本全国人口在下降，但东京的人口仍在上升。信息科技没有——至少在可预见的将来不会——强大到取代面对面交流的需求。
03
人力资本
人力资本外部性：低技能者受益更大
"人力资本外部性"——一个人的教育水平提高了，不仅自己收入提高，在与他人交往中还能促进知识传播，使别人的收入也提高。作者与哈佛大学格莱泽教授的研究用中国数据证实：一个城市的人均受教育年限增加一年，这个城市的劳动力平均工资可以获得21%的提高。更重要的是，低技能者所获得的人力资本外部性更大。 因为高技能者之间相互竞争会抵消一部分外部性的收益，而低技能者与高技能者有互补性。金融区需要保洁工，工程师需要操作工，高收入者需要餐馆服务员和家政——这就是劳动分工带来的技能互补。 大城市的政府不能认为自己不需要低技能劳动者了——恰恰相反，消费型服务业的从业人员大量是教育水平不高的人，而他们的工作又是不能被机器替代的。
04
就业创造
包容性的就业创造：实证证据
作者与高虹、佐藤宏使用CHIPS 2002年城市居民数据的研究：城市人口每增加100万人，个人就业概率平均提高0.66个百分点。 大学毕业生每增加100万人，就业概率提高4.3-6.6个百分点。更重要的是，就业增加效应对较低技能者更大：对受教育年数≤9年的劳动者，大学生增加100万→就业概率提高8.58个百分点；对9-12年者→3.57pp；对>12年者→仅2.12pp。为什么低技能者获益更多？ 在户籍制度下，各地落户政策偏向高学历者，导致高技能者在城市供给过多、竞争加剧、就业难。而高技能者集聚带来了对大量低技能劳动力的互补性需求——需求多、供给受制度限制，低技能者反而更容易找工作。

05
消费城市
消费型城市：服务业发展的关键
中国正在进入后工业化阶段，城市（特别是大城市）的重要性更强了。越是发达的国家，服务业在GDP中占比越高，城市化率越高，人口越向大城市集聚。原因在于人口密度有利于服务业发展。 与农业和工业相比，服务业的产品具有不可运输性，需要面对面完成。消费型服务业跟着人和钱走：人口密集的地方，固定投入（便利店、餐馆）更容易被分摊；劳动生产率越高的城市，富人越多，对消费型服务的需求越强。一个现代城市越发展高技术产业和生产型服务业，其带动的消费型服务业需求越多。中国第三产业占GDP比重从2001年的40.5%增长到2014年的48.2%，但比同等发展阶段的国家低了约10个百分点。 低密度的城市发展模式是制约服务业发展的重要原因之一。
06
城市模式
"活在城市"vs"生活在城市"：窄马路vs宽马路
通过对比上海浦西田子坊（窄马路、高密度）和浦东联洋社区（宽马路、低密度），作者揭示了两种城市发展模式的本质差异。田子坊每平方公里交叉路口多，沿街商业丰富，服务业密度高，居民靠步行和地铁出行，水果更新鲜更便宜。联洋社区宽马路、低密度，服务业配套不足，居民高度依赖开车出行，水果不太新鲜更贵。城市研究新词："宜于行走性"（walkability）。 窄马路适合人行走，沿街人口密度更高，服务业"进门人口"多，可以支撑商铺投资的固定成本。欧洲城市（巴黎、巴塞罗那、东京银座）每平方公里交叉路口逾百甚至200个，而上海浦东、北京城北仅十几个。城市规划不是路越宽越好——高密度、多小路反而减少通勤需求，引导服务业多样性和生活便利性。

产业控人
"以业控人"是南辕北辙
有些城市出于直觉，认为应该"以业控人"——淘汰低端产业、限制低技能者落户。但这违背了经济规律。那些来到城市的外来劳动力，绝大多数是因为城市对他们有需求才来的——他们的失业率远低于本地居民。产业升级了就不需要低端劳动力了？这是一个巨大的认识误区。一个城市的活力恰恰在于它的低端服务业。 高技能者和低技能者在城市中是"互补"的，有三个原因：劳动分工（每个生产单位都需要高低技能配套）、人力资本外部性（大学生也会对低技能者产生知识外溢）、消费外部性（高技能者产生大量生活服务需求）。更大的城市不是低技能者更少——恰恰因为高技能者带动了对低技能者的需求，更大的城市中体力型服务业的比重更高。
08
技能互补
高科技带动效应：一个岗位带动五个
美国的研究发现：城市中一个高科技产业的就业可以带动5个其他行业的就业岗位——2个相对高端（医生、律师）+ 3个消费型服务业（售货员、餐馆服务员）。制造业的带动力较弱，但一个制造业岗位仍然能带动1.6个本地服务业岗位。高低技能劳动力的比例大致为1:1。在中国，作者发现同样的规律：更大城市中从事体力型服务业的劳动者比重更高。然而，中国各个城市都给予大学以上文化程度者户籍优先权，导致高技能者在城市"供给过多"——大学生做银行柜面工作的"低层次就业"现象出现。而对低技能者的限制导致服务价格上涨（上海住家保姆月薪已超香港菲佣），高技能者的劳动生产率和生活质量反而因此下降。 一个定位为国际大都市的城市，要吸引全球人才，靠的是服务和生活环境，而不是人才公寓之类的硬件。

挑选后果
挑选劳动力技能的后果
如果城市政府通过户籍制度"挑选"高技能者，排斥低技能者，后果严重。对低技能劳动者数量的限制将减少体力型服务业的劳动供给，导致此类服务价格上涨。高技能人才收入高，必须大量消费服务——如果一个城市的服务价格太贵，它在全球人才竞争中的吸引力就会下降。同一个国家内部的城市之间，不应照搬一个国家对外国移民的政策。 在吸引高技能人才的同时限制低技能者进入，不仅妨碍公民的自由迁徙和居住权，而且城市会出现低技能劳动力短缺、工资上涨的怪现象。上海住家保姆月薪已超过香港菲佣——这是制度扭曲的鲜明信号。城市应该做的：通过技术和管理缓解"城市病"（增加基础设施和公共服务供给），而不是通过加重对外来人口的公共服务歧视来增加他们的生活成本。
10
政策方向
尊重市场：控制人口规模的正确方式
党的十八届三中全会报告先说"让市场成为配置资源的决定性因素，更好地发挥政府作用"，之后才说"要严格控制特大城市的人口规模"。这个次序是有含义的——任何控制人口规模的措施，必须尊重市场配置资源的基本规律，而不是逆转。控制人口的政策最多只能作为短期的应急措施（因为基础设施和公共服务不能一夜之间建成），而且只能适度缓解人口增长的速度，不可能让特大城市人口负增长。更不应该通过加重对外来人口的公共服务歧视来增加他们的生活成本。 从长期看，正确的政策只能：适应人口增长趋势，不断增加基础设施和公共服务供给，在不妨碍城市发展的情况下缓解拥堵和污染。而在经济增长缺乏动力的背景下，恰恰是这些缓解城市病的投资，才是拉动增长的有回报的投资。

🔑 核心结论
大城市不死。规模经济效应（分享、匹配、学习）和人力资本外部性使大城市在知识经济时代不可替代。实证研究表明，城市扩张能创造包容性就业，尤其有利于低技能劳动者——这正是因为高技能者和低技能者在生产和消费中具有天然互补性。一个高科技岗位能带动5个其他服务业岗位。"以业控人"和"挑选劳动力技能"的政策违反经济规律，会导致服务业价格上升、人才吸引力下降、城市活力受损。正确的城市发展模式是高密度、窄马路、适宜行走的"生活型城市"，而非低密度、宽马路的"活着型城市"。控制人口规模最多只能是短期应急措施——长期正确的政策是通过技术和管理降低"城市病"成本，同时不断增加基础设施和公共服务供给。大城市的活力，在根本上来源于人性。

故事一
密度与工资：人口越密集，工资水平越高
中国城市发展是全球趋势的又一例证。第一个故事：人口密度越大的城市，职工工资水平越高。 需要思考的是：如果只是因为大城市生活成本高，企业不会愿意多付工资。企业付高工资的原因一定是员工创造的价值更高——生产效率决定工资水平。在世界范围内，城市密度与劳动生产率之间始终呈现稳定的正相关关系。上海、北京等城市的人口密度虽高，但将其与同等城市中心距离的区域进行国际比较可以发现，上海的人口密度与东京、纽约、香港处于同一水平。中国城市发展的规律与世界其他国家并无不同，地理对城市发展的影响是决定性的。
02
故事二
人往高处走：大学生的集聚效应
第二个故事：大城市更"留得住"大学生。 数据显示，1982年人口普查时大学生比重较高的城市，到2010年大学生人口比重提高得更多——这和美国出现的大学生扎堆式集聚的现象如出一辙。一个城市平均受教育水平越高，该城市的工资水平就越高。与此同时，低技能劳动者也到大城市去寻找服务业工作，因此初始大学生比例高的大城市也出现了更快的总体人口增长速度。人力资本外部性使"人才集聚人才"成为自我强化的循环。 城市政策如果试图人为扭转这个趋势，将违背经济规律并付出代价。
03
故事三
港口与工资：地理的决定性影响
第三个故事：港口附近的企业集聚带来更高的劳动生产率。 在中国，到上海、香港或天津三大港口的最短距离显著影响工资水平。在500公里范围内，离港口越远，工资越低——相比港口附近，500公里外工资低约50%。500公里大致相当于从上海到安徽西部的距离，也就是泛长三角的范围。超过500公里后，工资水平与港口距离不再有明显关系。地理的影响是决定性的，经济政策要充分尊重经济规律。 如果和市场规律反向操作，不仅事倍功半，还可能事与愿违。
04
齐夫法则
齐夫法则与国家首位城市
根据齐夫法则（Zipf's Law）：一国最大城市的人口是第二大城市的2倍，是第三大城市的3倍……即最大城市人口是第N大城市的N倍。这一法则已被多国数据验证。改革开放前中国的计划经济人口迁移政策导致城市人口分散分布；改革开放后市场力量使中国城市人口分布越来越趋向齐夫法则。作者与陈钊教授为亚洲开发银行的研究对142个国家的数据分析发现：国家总人口可以解释首位城市人口85%的变动（R²=0.85）。 国家人口每增长1%，首位城市人口增长0.76%——人口向首位城市集聚的速度慢于国家人口增长，说明人们已将集聚成本纳入理性决策。根据这条拟合线，上海的位置偏小，而非偏大。

亚洲样本
亚洲国家的特殊性：人地关系紧张
亚洲国家人口密度普遍较高，首位城市的"城市病"更加严重。单独分析亚洲国家样本，拟合优度仍高达75%，但国家人口系数降至0.63——较高的人口密度使大城市地价和房价上升更快，"城市病"来得更快，阻滞了人口流入。然而，上海的位置仍然低于拟合线。 农业比重越高的国家人口分布越分散，工业和服务业比重提高时人口更加集中。这意味着随着中国城市化水平提高，还有大量农村人口要进城，其中一定有一部分要进入特大城市。根据模型，中国城市化进程结束时，上海人口预计约4100万——恰好居于之前估计的3000万至5400万之间。

东京参照
巨型城市的人中：东京道路的启示
作者把东京作为上海的主要参照城市，有三个原因：东亚国家人地关系紧张、相近的文化对服务业需求有类似影响、中国需要世界级都市圈。东京都的战后人口增长揭示了规律——1945-1965年人口每年增长37万；1920-1975年55年期间每年仍增长14.5万（东京都面积仅为上海的1/3）。按面积比例对应上海≈45万/年。 1990年代中期东京都人口短暂下降，与东京首都功能分散计划有关——但日本很快放弃该计划，因为行政人口疏散影响了东京的国际竞争力和整个日本的经济增长。这段历史值得中国反思。 东京都目前仍缓慢增长，而日本总人口在下降。全球国际大都市的人口都无一例外地在增长。

逆城市化
"逆城市化"的逆转：重回市中心
中国流行"逆城市化"的说法。虽然有数据表明北京和上海中心城区人口有所下降，但这不是长久趋势。东京的经验提供了警示：1965-1995年30年间，东京夜间人口占白天人口的比重直线下降；但从1995年以来，该比重又明显回升。人们重新搬回了市中心。 美国也出现了类似趋势——2010-2011年间，在面积前51位的美国都市区中有27个的中心城区人口增长超过郊区。原因包括：油价便宜的时期已过去，现代服务业发展增加了面对面交流需求，市中心犯罪率下降、环境改善、文体设施丰富。随着收入提高，时间成本上升——人们不愿再过着"不是在上班，就是在上班路上"的日子。中心城区的高房价正是人们为工作和生活便利性支付的成本。

上海预测
上海人口趋势：3000万至5400万之间
自1978年改革开放到2013年，上海年均人口增长（包括外来移民）约48万。考虑到上海面积是东京都的3倍，这并非"爆炸式增长"。上海非户籍人口增长近年来已随经济增长放缓而明显下降。关于2040年上海人口，复旦规划研究院有三种情景：情景一：保持现有人口不引进新劳动力→总人口降至1900万，60岁以上老人占比从15%升至41%（不可持续）。情景二：保持劳动年龄人口总量不变→总人口达3000万（仍不可持续）。情景三：保持劳动年龄人口比例不变→总人口达5400万（未考虑生活成本上升对人口的选择性挤出）。作者模型预测：中国城市化进程结束时，上海人口大约为4100万。关键态度：人口是经济发展结果，不是政府控制目标；科学预测并增加公共服务供给才对。

人口密度
人口密度不高于国际水平
有人以人口密度高为由主张限制城市规模。但比较到城市中心相同距离位置的人口密度，上海和东京、纽约、香港处于同一水平（参见图6-10），巴黎和伦敦反而更低。北京的人口密度比上海低更多。认为上海"人口密度太高"没有依据。 讨论人口密度时必须纠正两种错误比较：一是以全上海的人口除以总面积（上海面积大，这样算使密度显得小）；二是只拿内环线作比较（内环相当于其他都市核心区，会显得密度过高）。正确的比较方法是"到城市中心同样距离处的人口密度对比"，这才是客观的国际标准。

政策建议
尊重规律：科学预测而非主观调控
放眼全球，极少有国家像中国这样把城市人口作为政府调控指标——大部分国家把人口作为经济发展的结果，用作公共政策制定的预测目标。纽约预计未来还要增加100万人口；新加坡预计人口从550万增至700万。北京和上海总是提前很多年打破政府设定的人口控制目标，人们不去反思这些目标是否有误，反而断定"承载力已被突破"。一个简单的问题：如果承载力真的被突破，那现在超过承载力的人口是怎么被承载的？ 按照《上海市城市总体规划(1999-2020)》，2020年常住人口规划1800万左右，而2014年底实际已超2500万。正是这种滞后于实际人口增长的预测造成了公共服务紧张和基础设施短缺。科学的做法是：预测人口趋势，调整城市规划，增加公共服务和基础设施供给——而不是通过控制人口来控制需求。

——陆铭
🔑 核心结论
全球视野下的中国大城市发展并非特例。142个国家的数据揭示国家总人口是首位城市人口的主要决定因素（R²=85%），依此规律上海人口规模仍偏小。东京经验证明巨型城市在行政力量停止干预后人口持续增长，即使在国家总人口下降背景下亦然。上海2040年人口预计约4100万。城市管理者不应主观臆想人口目标——而应科学预测趋势，增加基础设施和公共服务供给。全球趋势正在逆转"逆城市化"，人们重新回到市中心，因为现代服务业需要面对面交流。

第15章 · 城市化之辩

本章是全书的第7章，逐一批判了关于城市化的五个常见误区。第一个误区——"中国人口过于集中"：数据显示中国城市间人口规模差距远低于国际水平，空间基尼系数仅0.4234（全球最低之一），绝大多数城市规模低于最优水平。第二个误区——"承载力极限"：城市人口一再突破规划，恰恰证明承载力可以在技术和管理进步下不断提升（青草沙水库解决水源、西电东输解决电力、地铁解决交通）。第三个误区——"市民化成本过高"：五种系统性高估导致成本被夸大，实际年度新增支出仅约6409亿元。第四个误区——"城市化与耕地矛盾"：农村建设用地面积是城镇的4倍，若进城农民宅基地复耕，耕地反而增加。第五个误区——"城市越大越污染"：高密度城市反而更环保（佛蒙特人均汽油是纽约市6倍），正确方向是"住得小、住得近、少开车"。

误区一
中国人口过于集中了吗？——不，恰恰相反
一个流行看法：过去30年大量资源向东部大城市集聚，所以中国集聚度太高了。数据给出的答案是否定的。宏观经济学家亨德森对142个国家2684个城市的统计：全球空间基尼系数0.5619，中国仅0.4234，为7个人口大国中最低。 日本最高（0.6579）。中国绝大多数城市规模低于最优水平——国际研究证实，受劳动力流动限制，中国城市需扩张而非缩小。中国城市间GDP规模差距在扩大，但人口和建成区面积的差距没有明显扩大。与中亚前计划经济国家相当。中国存在大城市不大、大城市数量不够多的规模均等化现象。 由于劳动力无法自由流动、建设用地指标受全国规划制约，中国的城市规模结构严重偏离市场均衡。
02
误区二
人口规模的承载力还是生命力？
北京历次人口规划总是提前多年被打破：1983年规划2000年控制1000万→1986年已达1000万；1991年规划2010年1250万→2000年已达1382万；2004年规划2020年1800万→2010年已达1961万。上海的规划也类似。城市人口规模一再突破"规划"，常被用作控制人口的依据——但鲜有人换个角度：今天的"城市病"是否正因规划人口远低于实际？ 所谓"承载力"面临的技术和管理因素都是可变的：青草沙水库供水量719万m³/天，解决1000万人用水；西电东输使上海不再缺电；日本东京轨道交通占80%客流。公共服务总量也是可变的——关键是投资。上海市2013年规划改用"预测"而非"控制"人口（2650万左右），这才是进步。城市人口不断突破"承载力"恰恰体现了城市的"生命力"。

误区三
市民化成本被严重高估——五种系统性错误
地方政府以"财政包袱"为由反对户籍改革，大量研究声称市民化成本高达10-14万元/人，总成本20-50万亿。作者指出五大严重误区：①忽视规模经济：市民化边际成本远低于平均成本（教室增加5名学生只需加座位）。②重复计算：市民化成本=城乡人均公共服务差额而非全部城市公共服务成本（城乡初中生均教育经费差额仅742元/年）。③以总支出替代年度流量：实际每年新增支出仅约6409亿元，而非20-50万亿。④混淆个人成本与公共成本："五险一金"主要由个人和企业缴纳，政府补贴很少（人均基本养老补助仅772元/年）。⑤忽视收益：市民化带来消费提升、治安成本下降、人力资本积累等巨大收益。真实市民化成本远没有预想的那么高。
04
误区四
城市化会遭遇耕地瓶颈吗？事实正好相反
误区：城市化推进→城市面积扩张→威胁耕地保护。 事实：2008年农村居民点占地16.53万km²，城镇建设用地仅4.06万km²——农村面积是城镇的4倍！农民进城后若将宅基地复耕，耕地总量反而增加。2001-2006年耕地减少主因是退耕还林（1998年华东大水后），而非城市化。英国二战以来城市化率提高较快，耕地没有减少；日本还略有增加。另一个误区："东部城市没地了"——打开百度地图卫星模式，上海仍有1990km²耕地，占全市面积1/3！即使在最密集的城市，还有大量低密度工业园可转为居住用地。以每平方公里5000人计，将上海耕地一半转为建设用地可容纳约500万人。这些土地出让收入和税收可创造约20万元/新进人口的财政空间。城市化不仅不威胁耕地，反而有利于耕地保护。

误区五
大城市=污染？挤挤更环保
人们凭直觉认为大城市更污染。戴维·欧文在《绿色城市》中揭示了一个悖论：纽约温室气体排放中79%来自楼宇（美国平均32%），纽约市副市长认为这说明问题——欧文反驳：这恰恰说明纽约环保！温室气体主要来自汽车尾气，纽约人开车少，所以楼宇排放占比显得高。对比佛蒙特州（人口密度26人/km²）和纽约市（密度10,000人/km²）：佛蒙特人均用电是纽约4倍，人均汽油是纽约市6倍！环保的出路是提高人口密度——"住得小、住得近、少开车"。 清华大学郑思齐研究发现：城市人口密度每平方公里增加1000人，每个家庭每年出租车和公交车碳排放分别减少0.424吨和0.837吨。上海20年前苏州河臭不可闻，今天空气质量超过华东绝大多数城市。环境恶化不是城市发展的结果，而是工业化必经的阶段。
06
宜居城市
何为宜居城市？——人性第一
EIU全球宜居城市排名中墨尔本、维也纳、温哥华前三，澳大利亚4城+加拿大3城进入前10——但这些国家人口密度极低（加拿大<3500万，澳大利亚<2300万），完全是国家人口密度的替代指标。新加坡国立大学的宜居指数更加"人性化"：分为经济活力、环境友好、安全稳定、生活质量和治理有效性五大类。前10中欧洲7席、亚洲2席（新加坡第3、香港第9）。中国的"北上广"分别排46、39、47位。注意：当今宜居城市（伦敦、东京）半个世纪前也曾污染严重。 改善环境靠产业结构调整——制造业转出、技术提升、生活方式绿色化、汽车使用量控制。正确道路不是倒退到原来环境好的贫困时代，而是进一步发展经济、调整产业结构、转变生活方式。宜居 = 发展和人性，不是反城市化。
07
养老金
谁为大城市养老？——外来人口是关键
出租车司机问："外来人口是不是太多了？"作者反问："你有没有想过，以后谁为你养老？"以上海为例：2011年底户籍60岁以上老人347.76万，占总人口24.5%；到2015年末突破30%。人均期望寿命已达82.29岁。养老金2010年余额-103.54亿元，2011年变为+211.61亿元——主因是社保新政使来沪就业人员综合保险转为城镇职工养老保险，参保人数从543万猛增至927万（增长70%）。扭转养老金入不敷出的地区性方法：提高劳动生产率和使人口年龄结构年轻化。如果按某些僵化想法保持上海常住人口不增长，很快将面临养老金入不敷出问题。 由于养老体系未全国统筹，人口流入地（上海、北京）因有大量外来年轻劳动力，实际人口年龄结构较好——上海劳动年龄人口中已有一半是外来人口。
"老师啊，你解开了我藏在心里5年的疑惑，过去我太狭隘了，只想到外地人来了抢我们的工作。现在我明白了……我们要想20年以后的事，要想以后上海怎么发展，以及谁来为我们养老的问题。"
——出租车司机

资金
农民工市民化的钱从哪儿来？
江苏北部某市文件：将农民转为市民需投入15-20万/人，10年转移160万人需2400-3200亿——政府财力无法支撑。这种困惑来自将"城市化"狭隘理解为"就地城市化"。如果农民可以异地城市化，将其宅基地复耕的建设用地指标带到高产值城市，问题迎刃而解。 政府现有支出效率也需评估：土地出让净收益仅1.6%用于廉租房（要求≥10%）；中西部基础设施投资已过剩；大量补贴农民建房（但中西部农村1/3住房是空的）。解决方案：蛋糕做大+建设用地指标跨地区交易。 重庆地票交易中，最高成交价超过30万元/亩——农民退掉土地可获得约25万元补偿。如允许跨省交易，补偿更高。中国要做的是政策突破+制度建设，而非禁止建设用地指标异地交易来回避问题。

建设用地指标的困境：填海造地与空心村

总结
走出城市化误区：结论与方向
综合全章分析，五个误区逐一澄清后，中国城市化的正确方向清晰可见：①人口再集中——中国的城市人口远不够集中，大城市不大、不多，需继续集聚以提高效率。②承载力可提升——技术和管理进步不断突破所谓"极限"，关键在于增加投资而非控制需求。③市民化不贵——年度新增成本约6409亿，相对于13万亿财政收入完全可以负担。④耕地不是问题——在合理制度下城市化反而有利于耕地保护。⑤环保靠密度——高密度城市人均碳排放更低，"拥挤"恰恰是绿色城市的特征。简·雅各布斯在1961年就已说透：对人口密度的追求不是反叛，而是回归人性。陶渊明说"心远地自偏"——偶尔厌倦大城市就去内心找安宁，而不是把城市建得像农村一样低密度。
"大城市的活力在根本上来源于人性。大多数人，在大多数情况下，还是害怕孤独的。"
——陆铭
🔑 核心结论
围绕城市化的五大误区——人口过度集中论、承载力极限论、市民化成本高论、耕地红线论、大城市污染论——逐一被事实和数据驳倒。中国城市间人口规模差距全球最低，绝大多数城市仍需扩张而非缩小。承载力在技术和管理进步下可不断提升。市民化真实年成本仅6409亿元而非20-50万亿。城市化不威胁耕地，农村面积是城镇4倍。高密度城市反而更环保。宜居城市的本质是发展经济和尊重人性，不是倒退到低密度的田园幻想。城市化的正确方向是集聚、开放与融合。

第16章 · 城市社会分割之困

本章是全书的第8章，深刻揭示了户籍制度造成的社会分割及其三重后果。第一重——农民工"健康耗损"：农民工进城时比城市居民更健康，但在脏、苦、险的环境中工作，在缺乏劳动保护（合同签订率仅41.3%）和社会保障的条件下，不健康的农民工在中年后返乡——城市留下了健康，农村承担了负担。第二重——消费损失：户籍制约使农民工消费比城市居民低约30%，相当于GDP的1.8%/年。第三重——身份与收入差距：与户籍身份相关的收入差距不仅伤害外来人口，也降低本地居民的快乐感，造成隧道效应中的"不能变道"式的社会不满。最后，作者从方言研究和文化融合的角度指出，全球城市的竞争最终是文化的竞争——上海话的"回报"恰恰说明融合的不足。

健康耗损
农民工留下青春和健康，带走自己
以打工诗人许立志的《失眠》开篇——"我们沿着铁轨奔跑，进入一个个名叫城市的地方，出卖青春，出卖劳动力……最后发现身上仅剩一声咳嗽，一根没人要的骨头。"农民工进城时平均比城市居民更健康（作者使用CGSS 2010数据分析），但从事"3D"工作——肮脏（Dirty）、辛苦（Demanding）、危险（Dangerous）。56.8%的农民工从事制造业和建筑业等重体力劳动，工伤和医疗保险参保率仅28.5%和17.6%。健康耗损效应使他们在进入中年后回到家乡，而留在城市的是相对健康的人群。 这就是"选择性"效应造成的假象——似乎留在城里的农民工并没有出现更严重的健康耗损。劳动合同保护缺失（合同签订率仅41.3%，建筑工人75.6%无合同）、"新农合"异地就医报销难（需多次往返家乡），使健康较差的农民工不得不返乡。城市消耗了他们的青春和健康，农村承担了不健康的负担。
02
歧视理性
歧视的原则：理性、效率、公平
作者以朋友刘伟伟的"太阳花"公益项目（为上海5-12岁进城务工人员子女提供教育资源）引出公共服务歧视问题。大城市通过教育权利限制（限制随迁子女入学）来"控制人口"。作者以三个原则分析：理性——劳动力迁移有两种动机：经济动机（更高收入）和公共服务动机（只为享受城市公共服务）。应该以就业和社会保障记录来识别后者，而不是根据职业、行业来主观遴选劳动力。"六小行业"整治（小餐饮、小理发等）打击的是城市居民真正需要的服务。效率——市场是配置资源的决定性力量，人口是经济资源的核心构成部分。政府不知道什么样的产业城市需要或不需要。公平——"对事不对人"：价格机制是最公平的，因为价格对每个人一样。政府要做的是调节收入差距本身，而不是用行政手段对外来人进行区分。"己所不欲，勿施于人"——如果你不想到小城镇去，就不要主张农民工就地城镇化。
03
消费损失
户籍制约的消费损失：≈GDP的1.8%/年
没有国家像中国这样——农业GDP已降至10%，农村人口仍占约一半。也没有国家有如此低的消费率：2009年最终消费/GDP仅48%，居民消费/GDP仅35%。中国独特之处在于户籍制度——像一个楔子钉进市场体制。 作者与陈斌开、钟宁桦的研究：一个没有所在城市户籍的外来居民消费比本地居民低约30%。按农民工总量测算，"消费损失"相当于当年GDP的1.8%——即每年GDP增长部分的约1/5。2007年数据验证差异缩小至16-20%，但农民工总量增长使其总消费损失仍然巨大。原因：不能安居乐业的预期制约耐用品消费；缺乏社会保障增加预防性储蓄；"返乡"预期使收入下降风险促使更多储蓄。更严重的是——限制外来人口消费最终也会损害本地居民的利益：梁文泉研究显示，外来人口汇款回家越多，所在城市消费型服务业的本地居民工资越低。

身份与快乐
为什么户籍不能成为身份？——隧道效应的启示
诺贝尔经济学奖得主阿玛蒂亚·森指出：当社会完全依照单一身份划分群体时，社会冲突增加。在中国，最重要的身份莫过于户籍。"隧道效应"比喻：你开车堵在隧道里，前面车道动了——你高兴（有希望了），这叫好的不平等。但若你所在的这条车道不能动，而边上车道动了——无法变道，你就愤怒了。与户籍身份相关的收入差距就是"不能变道"的不平等。 作者的研究：不同户籍身份的人均收入比值下降1，带来的快乐感提升相当于家庭人均收入提高53.2%——约等于城镇居民5年收入增长，或农民7.6年收入增长，或增加30m²住房面积。中国台湾"外省人"问题、印度的种姓制度、美国的肤色问题——都是身份变成不平等来源的教训。在中国，户籍造成的身份差异是制度造成的，本应更容易解决——如果持续得不到化解，对社会和谐的负面影响令人警醒。

收入差距
大城市的收入差距——全球普遍现象
大城市收入差距较高是全球普遍现象。作者使用2005年人口普查数据证实：越大城市的收入不平等程度越高。与美国不同，中国大城市收入差距主要由吸收大量外来人口驱动，且较低教育水平人群的收入差距更大——因为低教育者创业活跃（大学生创业反而不普遍），拉大了这一群体内部的收入分化。高技能者收入"与国际接轨"——要吸引全球人才就得支付全球水平薪酬。同时低技能者也向大城市集聚，因为高技能者创造大量服务需求。在大城市收入差距扩大几乎成为定局的条件下，不应直接干预市场力量决定的收入。 更长效的政策是用公共服务均等化缩小实际福利差距——以常住人口为对象。上海50.4%的"80后"外来人员分布在近郊区，30.9%在远郊区——公共服务布局与人口布局不匹配问题日益突出，各区中学人均资源差距达2-3倍。如果不尽早调整教育空间布局，未来矛盾更加尖锐。

文化融合
文化的融合——方言的回报与全球城市竞争力
作者与哈佛弗里曼、复旦陈钊、央财陈斌开共同研究方言对收入的影响。在中国的背景下，普通话是共同交流语言，本地方言主要不是信息交流功能，而是身份认同的信号发送机制。研究采用2005年1%人口抽样调查数据，将中国划为七大汉语方言区，发现说上海话能显著提高外来人口进入销售行业的概率——因为令本地客户产生信任感（"自己人"效应）。在上海，理发师来自苏北但说流利上海话，停车场看车人、钟点工都学会了上海话——不会说上海话的银行理财师反而流失本地客户。移民融入越好，本地方言越不应成为身份符号。 对比纽约（36%出生在国外、48%在家使用非英语语言）、伦敦（37%出生在国外、白种英国人已降至45%不到一半）——全球城市的人口由本地人、外地人和全球人才共同构成，城市文化也必须是多元文化的融合。随着制度不断市场化，人才的竞争最终成为文化的竞争——没有包容而多元的文化，全球城市的竞争力无从谈起。

城市新二元
"城市新二元社会"——居住分割的隐患
户籍制度不仅造成城乡分割，还在城市内部形成了"新二元社会"。大量研究发现城市内部本地户籍人口和外来人口之间收入差距明显，同等教育水平下本地户籍者收入更高。作者与陈钊团队的研究系系列发现：非户籍人口更不快乐，对政府、社会和邻里的信任度更低。针对上海的研究显示：非本地户籍人口出现聚居现象——即使在排除远郊居民和集体宿舍后仍然如此。外来居民居住条件明显更差，社区评价更低。更可怕的是代际遗传——上述分割将使外来人口下一代的成长环境和教育条件更差，贫困代际遗传，身份裂痕加剧。上海2014年常住人口约2500万，40%无本地户籍；珠三角部分城市超50%。如果这一群体持续扩大且收入低、不快乐、不信任、聚居——将给城市社会和谐带来巨大隐患。 与其用越来越多资源维护稳定，不如从现在开始消解"城市新二元结构"。

政策原则
城市管理：对事不对人
作者引用罗尔斯的"无知之幕"：讨论公共政策时应假设自己不知道自己的身份。中国智慧更接地气——"己所不欲，勿施于人"：有些人主张农民工就地城镇化或不要进特大城市——请问问自己，为什么你不去小城镇或农村？作为父母，如果政策让你的孩子与你分离——你怎么想？城市管理政策只能针对行为，不能针对人。短缺永远是价格太低时供小于求——如果在偏低价格下由政府规定谁有权利消费，才是不公平的。在绝大多数情况下，公正的市场经济也最有利于效率。中国面临两种效率损失：全局效率损失——每个地方维护本地利益，损失了跨地区资源配置效率；长期效率损失——百姓短视，官员任期也短，政策短期化。刘伟伟的评论："很多事情的发生有它的规律，我们要做的是尊重科学和规律，而不是以我们的好恶去看待它、以我们的利益去衡量它、以我们的权力去扭曲它。"

国际比较
全球城市的融合之路
全球化和城市化使人口跨地区迁移成为必然趋势，能否促进不同族群相互融合越来越重要。上海话/普通话/英语的混合——恰是移民城市充满活力的体现。 20世纪前半叶，上海曾吸引全球和全国的淘金者，上海话受苏州话、宁波话和英语的共同影响。未来，多种语言在上海融合的情形完全可能再现。中国一线城市都将成为"全球城市"——其人口由本地人、中国其他地方的新市民和全球优秀人才共同构成。城市文化也必须由传统的本地文化、中国其他地方的文化、国际上所有国家的优秀文化共同构成。西欧的教训：问题不是移民本身——问题是有移民、缺融合。移民带来互补性和多样性——文化和民族多样性有益于产生重大思想。 但如果融合不好，多样性可能转化为冲突。中国特大城市如何应对大量增长的国内和国际移民带来的影响、促进社会融合——必须立即提上议事日程，不能再拖。面对全球化的大时代，地域的文化虽然也要珍惜，但决不应该固守。

总结
社会融合：大国大城的终极挑战
从全章的六个维度——健康耗损、歧视原则、消费损失、身份与快乐、收入差距、文化融合——可以看出，户籍制度造成的社会分割是多维度、广谱性的。不仅伤害外来人口，最终也伤害本地居民。这不仅是一个公平问题——更是一个效率问题（消费损失相当于每年GDP的1.8%），更是一个社会和谐问题（身份相关的不平等降低所有人的快乐感）。解决方案清晰但不简单： ①公共服务逐步覆盖常住人口——不以户籍为门槛。②减少户籍与公共服务的挂钩程度——最终实现均等化。③促进社会和文化融合——让多元化成为城市活力而非冲突的源头。④以"对事不对人"的原则管理城市，用价格机制和市场力量引导资源配置。中国的城市管理者和居民都要思考：与其花更多资源维护稳定、雇用保安，不如从现在开始消解"城市新二元结构"——把这些资源用来做增进人民幸福的事情。在大国走向大城的道路上，社会融合是最后一道关——也是最难的一道关。

🔑 核心结论
户籍制度造成的"城市新二元社会"是当代中国面临的最深层挑战。农民工进城后留下青春和健康，带回不健康的身体——城市消耗了劳动力，农村承担了负担。户籍制约使居民消费损失达GDP的1.8%/年。与身份相关的收入差距降低所有人的快乐感——形成"不能变道"式的隧道效应。大城市收入差距是全球普遍现象，解决方案不是干预市场收入，而是公共服务均等化。随着制度的市场化，人才的竞争最终是文化的竞争——没有包容多元的文化，全球城市竞争力无从谈起。社会融合不是可选的附加项，而是大国大城必须直面的终极挑战。
