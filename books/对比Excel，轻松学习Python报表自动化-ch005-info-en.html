Excel vs Python Report Automation · Ch5「Setting Excel Alignment with Python」

SCQA · Situation → Complication → Question → Answer Adjusting alignment in Excel—left align, center, right align, text wrapping, merge cells—just takes a few clicks in the Alignment group on the Home tab. But when you have hundreds of rows that need uniform alignment, or daily generated reports needing standardized formatting, manual adjustment becomes tedious and error-prone. How do you batch-control Excel cell alignment with Python? This chapter systematically covers the 6 parameters of the Alignment() function, 9 horizontal/3 vertical alignment types, merge/unmerge cells, and a batch formatting case study.

This chapter focuses on the Python implementation of Excel's Alignment group on the Home tab. It first introduces the 6 core parameters of Alignment(): horizontal (9 types: left/center/right/justify/distributed/fill/general/centerContinuous/distributed), vertical (top/center/bottom), text_rotation (0-360°), wrap_text (auto-wrap), shrink_to_fit (auto-shrink font size), and indent (character indentation). It then covers merge_cells/unmerge_cells for merging cells, and styling merged cells via the top_left_cell reference. The chapter concludes with a batch alignment case study that uses a single for loop for table-wide standardized formatting.

Horizontal Alignment: 9 Types from Basic to Special
The horizontal parameter supports 9 types: left, center, right, justify (fills full line when enough text), distributed (auto-adjusts character spacing to fill), fill (repeats characters), general (default), centerContinuous, and distributed-indent. left/center/right are the most commonly used in daily work.

03
Wrap & Rotation
Auto-Wrap, Text Rotation & Font Auto-Fit
Three ways to handle text exceeding column width: wrap_text=True auto-wraps within the cell; shrink_to_fit=True auto-shrinks font to fit; text_rotation sets rotation angle (counterclockwise). These are extremely practical for long text titles and comment columns. Note: text_rotation and shrink_to_fit cannot be used simultaneously.

04
Merge Operations
Merge & Unmerge: merge_cells / unmerge_cells
merge_cells('A2:D2') merges specified range into one cell, equivalent to Excel's Merge & Center button. unmerge_cells('A2:D2') reverses the merge. The merged cell is treated as a single cell; styling is done through its top-left cell (top_left_cell), ensuring consistent formatting across the merged area.
from openpyxl.styles import Alignment, Font # Merge cells ws.merge_cells('A2:D2') # Style the merged cell via top-left cell top_left_cell = ws['A2'] top_left_cell.alignment = Alignment( horizontal='center', vertical='center') top_left_cell.font = Font(bold=True) # Unmerge ws.unmerge_cells('A2:D2')

🔑 Key Takeaway
Excel's alignment features—horizontal, vertical, auto-wrap, text rotation, indentation, merging cells—map to one core function Alignment() and two operators merge_cells()/unmerge_cells() in Python. Parameters directly mirror Excel ribbon buttons, making the learning curve minimal. The real power lies in batching: one for loop across all data cells, uniformly setting alignment properties, combined with a merged title row in bold. This is the core value of Python report automation—eliminating all repetitive, tedious work.
