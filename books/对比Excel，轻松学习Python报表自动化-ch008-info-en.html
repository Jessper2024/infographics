Excel vs Python Report Automation · Ch8「Editing Excel with Python」

SCQA · Situation → Complication → Question → Answer Excel's Editing group on the Home tab provides three core functions: sorting, filtering, and find & replace. Manually, you select a column → click Sort → choose ascending/descending, or click Filter → check conditions—repeat for every sheet. When data updates daily and requires re-sorting/filtering each time, this repetitive work is a productivity killer. How do you automate Excel sorting and filtering with Python? This chapter covers Pandas' sort_values() for single/multi-column sorting, Pandas conditional expressions for numeric and text filtering, and an overview of find & replace.

This chapter focuses on the Python implementation of Excel's Editing group on the Home tab. Key insight: openpyxl's sort and filter operations merely add sort/filter symbols—they do not actually reorder or filter data. Real sorting requires Pandas' sort_values(df, by=col, ascending=False), supporting both single-column and multi-column sorting (pass a column name list, set ascending per column). Real filtering uses Pandas conditional expressions (e.g., df[df['col1']>2] for numeric, df[df['col2']=='b'] for text), which are far more flexible than Excel's mouse-driven interface. Find & replace is covered in Chapter 9. This chapter demonstrates the Python ecosystem's "compose together" philosophy—openpyxl handles Excel file I/O, Pandas handles data analysis logic.

01
Sorting
sort_values(): Single & Multi-Column Sorting in One Step
openpyxl's sort operation only adds sort symbols (Excel's dropdown arrows) without actually reordering data. Real sorting needs Pandas: df.sort_values(by='col1', ascending=False) sorts col1 descending. For multi-column: by=['col1','col2'], sorting col1 first then col2 for ties, with per-column ascending settings.
import pandas as pd # Single-column sort: col3 descending df_sorted = df.sort_values(by='col3', ascending=False) # Multi-column sort: col1 desc, ties by col2 asc df_sorted = df.sort_values( by=['col1', 'col2'], ascending=[False, True])
02
Filtering
Pandas Conditional Filtering: Numeric & Text Dual Modes
openpyxl's filter also only adds filter symbols without truly filtering data. Pandas uses conditional expressions directly: df[df['col1']>2] filters rows where col1>2; df[df['col2']=='b'] filters where col2 equals 'b'. Excel separates numeric and text filtering modes; Pandas unifies them with one expression syntax, supporting complex combined conditions (&, |).

🔑 Key Takeaway
Excel's sorting and filtering require Pandas to achieve true data reordering and filtering in Python. openpyxl handles Excel file I/O; Pandas handles data analysis logic—a classic example of Python's "compose libraries together" philosophy. sort_values() supports single and multi-column sorting; conditional expressions (df[condition]) handle both numeric and text filtering, far more flexible and powerful than Excel's mouse-driven approach. Sort, filter, format, find & replace form a complete workflow; write the script once, run it daily—this is the ultimate form of Python report automation.
