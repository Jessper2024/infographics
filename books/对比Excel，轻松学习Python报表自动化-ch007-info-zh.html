对比Excel，轻松学习Python报表自动化 · 第7章「用Python设置Excel单元格」

SCQA · 情境 → 冲突 → 问题 → 回答 Excel工作表的日常维护离不开插入行、删除行、调整行高列宽、隐藏行列。手工操作时，右键→插入→右键→删除→右键→行高→右键→列宽——重复无数次。当报表有几十个Sheet且每个都需要相同调整时，手工操作不仅耗时，还容易错删误改。如何用Python自动化管理Excel的单元格行列结构？本章讲解insert_rows/insert_cols、delete_rows/delete_cols、行高列宽设置（row_dimensions/column_dimensions）、行列隐藏（group组合），以及批量遍历设置整表的实战案例。

本章聚焦于Excel「开始」选项卡「单元格」组的Python实现，涵盖四大操作：①插入行/列——insert_rows(m, n)和insert_cols(m, n)，m为位置，n为数量（默认1）；②删除行/列——delete_rows(m, n)和delete_cols(m, n)，参数含义相同，是插入的逆向操作；③行高/列宽设置——通过row_dimensions[row_num].height和column_dimensions[col_letter].width分别设置；④行列隐藏——通过column_dimensions.group和row_dimensions.group创建组合来实现（Python中无直接隐藏API，用组合方式更灵活，支持折叠/展开）。最后通过一个批量案例展示如何遍历多行多列统一设置行高列宽，实现整表格式标准化。

01
插入操作
insert_rows / insert_cols：动态扩展工作表结构
insert_rows(m, n)在第m行前插入n行，insert_cols(m, n)在第m列前插入n列。n默认值为1，可省略。这是Excel右键「插入」功能的Python等价操作。插入后原有数据自动移位，可复制原Sheet对比插入前后效果。适用于动态添加数据区域、预留空行、插入分隔行等场景。

02
删除操作
delete_rows / delete_cols：精准删除多余行列
delete_rows(m, n)从第m行开始删除n行，delete_cols(m, n)从第m列开始删除n列。n默认值为1，可不写。这是插入的逆向操作，对应Excel右键「删除」命令。用于清理空行、删除不需要的数据列、精简报表结构。注意：删除操作不可逆，执行前建议先备份原Sheet。

03
尺寸设置
行高与列宽：row_dimensions / column_dimensions
行高通过ws.row_dimensions[row_num].height设置，列宽通过ws.column_dimensions[col_letter].width设置。这与Excel右键「行高」「列宽」对话框完成相同的功能。调整其他行只需换行号数字，调整其他列只需换列字母即可，简单直观。
# 设置第1行行高为40，第A列列宽为20 ws.row_dimensions[1].height = 40 ws.column_dimensions['A'].width = 20 # 批量设置：遍历行号/列字母 for r in [1, 2, 3]: ws.row_dimensions[r].height = 40 for c in ['A', 'B', 'C']: ws.column_dimensions[c].width = 20
04
隐藏操作
行列隐藏：通过组合（group）实现可折叠区域
openpyxl中隐藏行列的方式是创建组合（group）而非直接的隐藏API。ws.row_dimensions.group(7, 10)隐藏第7-10行，ws.column_dimensions.group('D', 'F')隐藏D-F列。这种方式比单纯的隐藏更灵活——支持折叠/展开，用户可在Excel中手动展开查看。

批量实战
批量设置多行/列的行高列宽：遍历整表标准化
插入/删除行列通常是一次性操作，无需循环。但行高列宽往往需要整表统一设置。通过遍历行号或列号列表，在for循环中对每一行/列分别设置height/width，即可一次性完成整表的尺寸标准化。代码编写一次，永久适用于同类报表。

🔑 核心结论
Excel的行列结构管理——插入、删除、调整行高列宽、隐藏——在Python中对应四对函数/属性：insert_rows/insert_cols、delete_rows/delete_cols、row_dimensions.height/column_dimensions.width、group组合隐藏。这些API直接映射Excel的右键菜单功能，学习曲线平缓。实际工作中，插入删除通常是一次性操作，而行高列宽则需要批量遍历整表设置。一个for循环遍历所有目标行列，统一设置height和width，让整个工作表的布局一步到位。Python让Excel的「体力活」变成了「智力活」。
