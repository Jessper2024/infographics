对比Excel，轻松学习Python报表自动化 · 第8章「用Python对Excel进行编辑」

SCQA · 情境 → 冲突 → 问题 → 回答 Excel的「开始」选项卡「编辑」组提供排序、筛选、查找替换三大核心功能。手工操作时，选中列→点击排序→选升序/降序，或点击筛选→勾选条件——每个Sheet重复一遍。当数据每天更新、每次都要重新排序筛选时，这些重复操作就是生产力杀手。如何用Python自动完成Excel的排序与筛选？本章讲解用Pandas的sort_values()实现单列与多列排序、用Pandas条件表达式实现数字筛选与文本筛选、以及查找替换的概要介绍。

本章聚焦于Excel「开始」选项卡「编辑」组的Python实现。核心要点：openpyxl的排序和筛选只会添加排序/筛选符号，不会真的改变数据顺序或过滤数据。真正的排序操作需使用Pandas库的sort_values(df, by=col, ascending=False)，支持单列排序和多列排序（列表传多列名，分别设升/降序）。筛选操作使用Pandas条件表达式（如df[df['col1']>2]实现数字筛选，df[df['col2']=='b']实现文本筛选），比Excel鼠标操作更灵活强大。查找替换在第9章详细讲解。本章展示了Python生态「组合使用」的核心思想——openpyxl处理Excel文件读写，Pandas处理数据分析逻辑。

01
数据排序
sort_values()：单列与多列排序一步到位
openpyxl的排序操作只添加排序符号（即Excel界面的下拉箭头），不会真正重排数据。真正的排序需用Pandas的sort_values()：df.sort_values(by='col1', ascending=False)按col1降序。多列排序传列名列表：by=['col1','col2']，先按col1排，相同值再按col2排，可分别设升/降序。
import pandas as pd # 单列排序：按col3降序 df_sorted = df.sort_values(by='col3', ascending=False) # 多列排序：按col1降序，相同值按col2升序 df_sorted = df.sort_values( by=['col1', 'col2'], ascending=[False, True])
02
数据筛选
Pandas条件筛选：数字与文本双模式
openpyxl的筛选同样只添加筛选符号，不真正过滤数据。Pandas用条件表达式直接筛选：df[df['col1']>2]筛出大于2的行，df[df['col2']=='b']筛出等于b的行。Excel分为数字筛选和文本筛选两种模式，Pandas的条件表达式统一处理，且支持复杂组合条件（&、| 多条件联合）。

从Excel到排序结果：完整的数据处理链路
完整的数据排序筛选流程：①用pandas.read_excel()读取Excel数据到DataFrame；②用sort_values()按指定列排序（支持多列、混合升/降序）；③用条件表达式筛选目标数据；④用to_excel()将结果写回Excel或直接用于后续分析。这一链路将Excel一整套手工操作彻底自动化。

04
对比分析
Excel vs Python 排序筛选对比
Excel的排序筛选是手动交互式操作——每次数据更新要重新点一遍，多列排序需弹窗设置。Python的排序筛选是代码驱动——数据变化时只需重新运行脚本，无需任何手动操作。Pandas的条件筛选远比Excel灵活：可组合任意复杂条件、可在排序前先筛选、可直接链式操作（筛选→排序→取top10）。

🔑 核心结论
Excel的排序与筛选在Python中需要借助Pandas库来实现真正的数据重排和过滤。openpyxl负责Excel文件读写，Pandas负责数据分析逻辑——这是Python生态「组合使用」思想的典型体现。sort_values()支持单列和多列排序，条件表达式（df[条件]）支持数字和文本两种筛选模式，比Excel的鼠标操作更灵活、更强大。排序、筛选、格式化、查找替换串联成完整工作流，一次编写脚本，每天运行即可——这就是Python报表自动化的最终形态。
