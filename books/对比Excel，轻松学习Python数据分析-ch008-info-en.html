Arithmetic & Comparison Operations

Arithmetic operations are the most basic calculations in data analysis; in Python, numeric columns can be directly operated on:

  • Column vs column: df['A'] + df['B'] (all four basic operations supported, element-wise)
  • Column vs constant: df['A'] + 10 (entire column uniformly add/subtract/multiply/divide)

Comparison operations return boolean values for data filtering:

  • Standard comparison operators: >, >=, ==, !=, <, <=
  • Results serve as boolean index conditions — the core mechanism for data filtering
2
Central Tendency Metrics

Four core metrics in aggregation operations reflecting central tendency:

  • Mean (mean): arithmetic average, reflects general level, greatly affected by extreme values
  • Median (median): middle value after sorting, not affected by extreme values, more representative of distribution than mean
  • Mode (mode): most frequent value, reflects the most common level
  • Quantile (quantile): finer position-based metric than median, including Q1 (25th), Q2 (50th=median), Q3 (75th) percentiles

In Python, call corresponding functions directly; axis=0 (default) by column, axis=1 by row.

3
Dispersion Metrics

Metrics measuring data volatility, complementary to central tendency metrics:

  • Variance (var): basic measure of data dispersion; larger values indicate greater volatility
  • Standard Deviation (std): square root of variance; same units as original data, more intuitive
  • Range: max() - min(), the difference between maximum and minimum values

Variance and std are often used with mean: mean reflects level, std reflects stability. Two datasets with same mean but different std indicate very different stability.

4
Count & Sum Operations

The most basic aggregation operations, also the most frequently used:

  • Non-null count (count): count non-empty values; Excel distinguishes counta() (non-empty) and count() (numeric); Python uses count() uniformly
  • Sum (sum): calculate total of area values; Python's sum() defaults to column-wise, axis=1 for row-wise
  • Max/Min (max/min): return extreme values in area; commonly used for initial outlier detection

count() is the first step in data quality checking: the difference between non-null count and total rows equals the number of missing values.

5
Correlation Analysis

Correlation measures the degree of association between two variables, forming the basis for deeper analysis:

  • Pearson correlation coefficient: most commonly used; range [-1, 1]; positive = positive correlation, negative = negative correlation, 0 = no linear relationship
  • Excel: CORREL() function, specify two ranges
  • Python: corr() method; call on two columns for single value; call on entire DataFrame for pairwise correlation matrix across all fields

The correlation matrix is an important foundation for multivariate analysis, quickly revealing hidden relationships between variables—the mathematical backbone of "beer & diapers" type association analysis.
