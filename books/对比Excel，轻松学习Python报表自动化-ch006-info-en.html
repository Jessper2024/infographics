Excel vs Python Report Automation · Ch6「Setting Number & Conditional Formats with Python」

SCQA · Situation → Complication → Question → Answer Excel's number and conditional formatting are powerful data visualization tools—showing 0.5 as 50%, dates as YYYY-MM-DD, highlighting out-of-range cells in red, adding color scales or data bars. Manually, you select each range, click the conditional formatting menu, and set rules. But when reports update daily with large datasets, repeating all this manually is exhausting. How do you automate Excel's number display formats and conditional formatting with Python? This chapter systematically covers number_format's 9 number formats, CellIsRule highlighting, DataBarRule data bars, ColorScaleRule color scales, and IconSet icon sets.

This chapter is divided into two major sections: number formats and conditional formatting. The number_format property corresponds to Excel's Number group, covering 9 common formats: General, Decimal (0.00), Percentage (0.00%), Currency (¥#,##0), Scientific (0.00E+00), and DateTime (yyyy-mm-dd h:mm:ss). The conditional formatting section covers four types: CellIsRule() for highlighting cells meeting conditions (supports 7 operators: greaterThan, lessThan, between, equal, etc.); DataBarRule() for data bar visualization (customizable min/max types, values, colors, showValue toggle); ColorScaleRule() for color scales (two-color and three-color gradients); and IconSet() for icon sets (multiple icon types with reverse and showValue options). Together they form the Python automation solution for Excel data visualization.

02
Highlighting
CellIsRule: Auto-Mark Abnormal Data by Condition
conditional_formatting + CellIsRule(operator, formula, fill) implements standard conditional formatting. The operator parameter supports greaterThan, lessThan, between, equal, notEqual, and more. formula specifies thresholds (use a list like [50,80] for between). fill uses PatternFill for color. Commonly used to flag over-budget spending, under-target KPIs, or data within acceptable ranges.
from openpyxl.formatting.rule import CellIsRule from openpyxl.styles import PatternFill # Highlight cells greater than 50 in red ws.conditional_formatting.add('A1:A10', CellIsRule(operator='greaterThan', formula=['50'], fill=PatternFill(fill_type='solid', fgColor='FF0000'))) # Highlight cells between 50-80 ws.conditional_formatting.add('B1:B10', CellIsRule(operator='between', formula=['50','80'], fill=PatternFill(fill_type='solid', fgColor='FFFF00')))
03
Data Bars
DataBarRule: Visualize Data Magnitude at a Glance
DataBarRule(start_type, start_value, end_type, end_value, color, showValue) displays values as data bars. start_type/end_type can be min/max/num/percent/percentile/formula. color sets the bar color. showValue controls whether numbers are displayed. Extend to multiple columns by changing the range to 'A1:B10'.

ColorScaleRule: 2-Color and 3-Color Gradient Comparison
ColorScaleRule(start_type, start_value, start_color, mid_type..., end_type...) uses color gradients to visualize value distributions. Two-color scale uses only start+end params; three-color scale adds a midpoint. Commonly used for heatmaps, score distributions, and temperature data visualization.

Icon Sets
IconSet: Display Data Tiers with Icons
IconSet(iconSet, percent, cfvo, showValue, reverse) maps data to icons. iconSet specifies the icon type (3Arrows, 3Flags, 3TrafficLights, etc.). The cfvo list defines threshold values for each tier. showValue toggles numeric display. reverse flips icon direction. Like phone signal bars, icons make data tiers instantly visible.

🔑 Key Takeaway
Excel's number and conditional formatting are powerful tools for data presentation. In Python, the number_format property covers 9 common number formats including General, Decimal, Percentage, Currency, and DateTime. Conditional formatting is fully reproduced through four rule functions—CellIsRule (highlighting), DataBarRule (data bars), ColorScaleRule (color scales), and IconSet (icon sets)—whose parameters directly mirror Excel dialog options with a clear learning path. Batch application simply requires changing the range from 'A1:A10' to 'A1:C20', applying one rule to multiple columns. With both number formatting and conditional visualization fully automated, report readability and professionalism dramatically improve.
