Python简介与环境搭建

Python是一门语法简单、库丰富的胶水语言，2018年超越Java成为编程语言排行第一。本书选择Anaconda发行版作为入门环境，原因是它自带常用库，无需手动安装，对初学者极其友好。

  • Anaconda安装流程：确认系统位数 → 官网下载 → 选择Python 3版本 → 双击安装 → 默认路径即可
  • IDE vs IDLE：IDE（集成开发环境）包含编辑器+编译器+调试器；Jupyter Notebook是数据分析领域最常用的IDE
  • Jupyter Notebook核心操作：新建Python 3文件、运行代码（Ctrl+Enter）、重命名（File>Rename）、保存/另存为（Download as）、导入文件（Upload）
2
Jupyter Notebook进阶：Markdown与目录

Jupyter Notebook不仅是代码编辑器，更是数据分析的交互式笔记本。两大进阶功能极大提升工作效率：

  • Markdown模式：将代码框切换到Markdown模式后变成文本框，支持Markdown语法。数据分析时可直接在代码间写下分析结论，实现"代码+文档"一体化
  • 目录功能：非自带，需通过Anaconda Prompt执行 pip install 安装。安装后可通过目录快速跳转到不同代码段，类似书籍目录和毕业论文目录
3
七大数据类型与操作

Python的七种核心数据类型覆盖了数据分析中所有常见的数据组织形式：

  • 数字 — 整数(int)、浮点数(float)。支持加减乘除、乘方(* *)、取整(//)、取余(%)
  • 字符串 — 连接(+)、复制(*)、长度(len)、查找(in/find)、索引(普通/切片)、分隔(split)、去空格(strip)
  • 列表 — 有序可变集合。新建、复制(*)、合并(+或extend)、插入(append/insert)、计数(count)、位置(index)、索引、删除(pop/remove)、排序(sort)
  • 字典 — 键值对结构{key:value}。键必须唯一。keys()/values()/items()三个核心方法
  • 元组 — 有序不可变集合，用小括号。与列表可互转(list()/tuple())。zip()函数将多个可迭代对象打包成元组
4
运算符与流程控制

四类运算符与三种控制流是Python程序逻辑的核心：

  • 算术运算符：+、-、*、/、* *（乘方）、//（整除）、%（取余）
  • 比较运算符：==、!=、>、<、>=、<=，返回True/False
  • 逻辑运算符：and（与）、or（或）、not（非）
  • for循环 — 遍历序列（列表/字符串），逐个执行。如：依次学习Excel→SQL→Python→统计学
  • while循环 — 条件满足时持续执行。如：学习时间＜7周→继续学；≥7周→找工 作
  • 条件语句 — if/else/elif。多条件判断时elif语句逐级判断，最多执行一个分支
5
函数与高级特性

函数是一段可重复使用的程序。掌握普通函数与匿名函数两种形式，以及列表生成式和map等高级特性：

  • 普通函数：def 函数名(参数): + 语句块 + return。形参在定义时声明，实参在调用时传递
  • 匿名函数(lambda)：lambda arg1,arg2: expression。比普通函数简洁，常用于简单操作（如两数相加→ lambda x,y: x+y）
  • 列表生成式：一行代码替代嵌套for循环。如 [x*y for x in listA for y in listB]，数据量大时显著提升速度
  • map函数：map(function, args)，对序列每个元素执行function操作。结果需for遍历或list()转换才能显示
  • 模块：升级版函数，跨程序可调用。数据分析三大核心模块：NumPy、Pandas、matplotlib。import 模块名 或 from 模块 import 函数
