Missing Value Handling

Missing values are one of the most common data quality issues. Three main processing strategies:

  • View missing values: Excel uses count comparison + Go To Special (Ctrl+G); Python uses info() and isnull() methods for quick detection
  • Delete missing values: Excel uses Go To Special → Delete entire row; Python uses dropna() (default deletes rows with NaN), how='all' only deletes fully empty rows
  • Fill missing values: Excel uses Go To Special + Ctrl+Enter for batch fill; Python uses fillna() supporting global fill, column-specific fill, forward/backward fill

Principle: Fill rather than delete when missing rate is below 30% — data is valuable.

2
Duplicate Value Processing

Duplicate records refer to rows with identical data, typically handled by deletion.

  • Excel: Data → Data Tools → Remove Duplicates; can check all fields or specified fields; keeps first record by default
  • Python: drop_duplicates() method; supports dedup by specified columns (subset parameter); keep='first'/'last'/False controls retention strategy

Typical scenario: deduplicate by unique ID to count customers; sort by time in ascending order before deduplication to retain earliest records.

3
Outlier Detection & Handling

Outliers are values significantly higher or lower than normal data. Three detection methods:

  • Business experience: define normal range based on domain rules
  • Box plot: points above the upper fence or below the lower fence are outliers
  • 3σ rule: when data follows normal distribution, values deviating from the mean by more than 3 standard deviations are outliers

Handling: most commonly deletion, or treat as missing values and fill, or investigate causes separately. Python uses filtering to delete outliers (df[df['age']<200]) and replace() for filling replacement.

4
Data Type Conversion

Different data types enable different operations; must convert to target type before analysis.

  • Excel: Select column → Number format dropdown → choose target type
  • Python: astype() method, e.g., df['age'].astype('float'); Pandas has 6 main data types (int/float/object/string/bool/datetime)
5
Index Management

Indexes are the basis for data lookup, like organizing food in refrigerator zones for quick access.

  • Add index: Excel inserts rows/columns; Python sets columns and index parameters
  • Reset index: Excel drags column to first position; Python uses set_index(), supports hierarchical indexing (multi-column combination)
  • Rename index: Excel directly edits field names; Python uses rename()
  • Reset to default: Python uses reset_index() to convert hierarchical index back to regular columns, commonly used in grouping and pivot tables
📊 Chapter Overview

Data preprocessing is the foundation of analytical work. Dirty data leads to "garbage in, garbage out." Mastering the three core operations—missing value filling/deletion, duplicate removal, and outlier detection—along with data type conversion and index management, transforms raw data from "market ingredients" into "kitchen-ready produce" for subsequent analysis.
