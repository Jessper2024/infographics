1
Value Replacement & Sorting

Value replacement is a common data cleaning tool, with three replacement modes:

  • 1-to-1 replacement: replace(A, B) replaces A with B (Excel uses Ctrl+H)
  • Many-to-1 replacement: replace([A,B], C) replaces multiple values with one (Excel needs OR+IF functions)
  • Many-to-many replacement: replace({"A":"a","B":"b"}) dictionary-style batch replacement (Excel needs multiple IF nesting)

Value sorting uses sort_values(), supporting single/multi-column sorting, ascending/descending, and missing value position control (na_position). Excel uses the Sort & Filter menu.

2
Value Ranking & Deletion

Ranking (rank()) adds a ranking column; the method parameter controls duplicate handling:

  • average: average rank for duplicates (corresponds to Excel RANK.AVG)
  • first: rank by order of appearance
  • min: best rank for duplicates (corresponds to Excel RANK.EQ)
  • max: worst rank for duplicates

Value deletion (drop()) supports deleting columns (axis=1), rows (axis=0), and conditional deletion (filter to keep non-matching rows as new data source).

3
Counting, Unique Values & Search

Three high-frequency query operations, each with clear tool mapping:

  • Value counting: Excel uses COUNTIF(); Python uses value_counts(); normalize=True for proportions
  • Unique value extraction: Excel copies → remove duplicates; Python uses unique() method in one step
  • Value search: Excel uses Find dialog (Ctrl+F); Python uses isin() method, supporting single and multiple value search
4
Binning & Row/Column Operations

Binning divides continuous values into groups — Excel requires multiple IF nesting; Python uses cut() (specify bins) and qcut() (specify number of groups, auto-equal-size).

Row/column insertion: Excel right-click insert; Python uses insert() to specify position and column name.

Transpose: Excel Paste Special → Transpose; Python uses .T method.

Index reshaping: stack() converts tabular data to tree form (column index becomes row index), unstack() does the reverse — a Python-exclusive advanced feature.

5
Wide-Long Conversion & Apply Functions

Wide-long conversion is a core data restructuring operation:

  • Wide → Long: stack() (requires setting index first) or melt() (id_vars keeps unchanged columns, var_name/value_name set new column names)
  • Long → Wide: pivot table pivot(), setting row index, column index, and value column

apply() and applymap(): Execute the same function on DataFrame row/column elements, used with lambda anonymous functions. apply for columns/rows, applymap for each element — advanced Pandas batch data processing features.

📊 Chapter Overview

Value operations are the "knife skills" of data analysis. From replacement and sorting to wide-to-long conversion, 13 operations cover the vast majority of daily data processing needs. The key distinctions: replace for cleaning, sort_values for ordering, rank for ranking, drop for deletion, melt/stack for reshaping. In Python, most operations require just one line of code—far more efficient than Excel's nested functions, which is Python data analysis's core advantage.
